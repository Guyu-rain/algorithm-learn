# C++ 中的红黑树

C++ STL 中底层用红黑树实现的是 `set` 和 `map`。（或者 `multiset`, `multimap`）。前者是`TreeSet`

由于是通过红黑树实现，因此这两个容器都自然有序 `set` 是按照 `key` 排序。

## lower_bound & upper_bound

红黑树的成员方法 `lower_bound` / `upper_bound` 的时间复杂度是 $O(logN)$。

返回值都是迭代器`std::map::iterator<K, V>`，含义都是**找上界**：

- map.lower_bound(val)：找`set`中`key`**不小于**`val`的第一个元素。找到的是大于等于`val`的上界；
- map.upper_bound(val)：找`set`中`key`**大于**`val`的第一个元素。找到的是大于`val`的上界；

如果找不到，返回 `map.end()`。

如果是**区间检测**，假设我们用`map<startTime, endTime>`我们有一个区间 `[start, end)`，要检查和其他区间是否有重叠，显然我们要检测`next interval`和`prev interval`。

我们通过startTime来排序，通过endTime来确定是否有冲突。一般而言，我们将开始时间相等化为`next interval`的情况，我们只需要找`next interval`，然后前一个就是`prev interval`，可以这样来判断：

```C++
map<int, int> map;

bool haveIntersect(int start, int end) {
    // 找 next interval
    auto it = map.lower_bound(start);
    // next interval，比较结束时间
    if (it != map.end() && it->first < end) {
        // 如果有 next interval 且开始时间比 end 早
        // 有交叉
        return true;
    }

    if (it != map.begin() && prev(it)->second > start) {
        // 如果有 prev interval 且结束时间比 start 晚
        // 有交叉
        return true;
    }

    return false;
}
```
