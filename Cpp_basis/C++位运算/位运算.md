# C++中的位运算
位运算只能作用于整数类型上，(`int`, `char`, `unsigned int`, `long long`...)

### 有如下运算符：
- `&`: 按位与
- `|`: 按位或
- `^`: 按位异或
- `~`: 按位取反
- `<<`: 左移
- `>>`: 右移

### 一般使用方式：
#### 高效二进制乘除法
```C++
a << n; // 等价于 a * pow(2, n)
a >> n; // 等价于 a / pow(2, n)
```

#### 判断奇偶性
```C++
(a & 1) == 0; // 偶数
(a & 1) == 1; // 奇数
```

#### 切换某一位
```C++
a ^ (1 << n); // 将第n位反转
```

#### 交换某一个数
```C++
a ^= b;
b ^= a;
a ^= b;
```
