# C++ 字符串处理

## 字符串流对象
在头文件 `stringstream` 中，提供了将字符串当作流处理的类；
- `std::stringstream`：读写
- `std::istringstream`：读
- `std::ostringstream`：写

## 读取字符串
- 读取一个单词，遇到空字符(空格，换行符，制表符...)截止。
```C++
string s;
cin >> s;
```
> 注意⚠️：cin 不会消耗空字符，cin 字符串的行为是读取一整个字符串，遇到空字符停止但是不消耗空字符。
> 比如，用户输入：
> ```shell
> Hello
> world
> ```
> 缓冲区内容为：`Hello\nworld\n`。cin 的行为是读取 `Hello`，但是不消耗`\n`。因此留下的内容是`\nworld\n`
- 读取整行，遇到换行符截止。
```C++
string s;
getline(cin, s); // 等价于 getline(cin, s, '\n');
```
> 注意⚠️：getline 默认读取到换行符，而且会消耗换行符。
> 如果缓冲区有`Hello\nworld\n`，然后使用如下代码来读：
> ```C++
> cin >> x;
> getline(cin, s);
> ```
> x 读完后，缓冲区内容为`\nworld\n`，然后紧接着`getline()`读取的内容是`\n`，读完缓冲区剩余`world\n`

- 一个方法：`cin.ignore()`
默认为 `cin.ignore(1)`，丢弃输入流的一个字符。

## 字符串 split
C++ 并没有内置 split

- 按照**指定规则**读取字符串
```C++
string s;
istringstream iss(s);
// 按照 , 分隔
string word;
while (getline(iss, word, ',')) {
    cout << word << endl;
}
```

- 特别的，如果要按照空格等空字符来split，`std::istringstream` 的默认行为就是按照空字符split
```C++
string s;
istringstream iss(s);

string word;
while (iss >> word) {
    cout << word << endl;
}
```

## 删除空格
- 删除字符串中所有空格，或者某个指定字符
```
s.erase(remove(s.begin(), s.end(), ' '), s.end);
```
注意：这里的 `remove(first, last, val)` 方法是标准库中的一个方法，含义是将`[first, last)`内不等于 val 的元素向后搬，然后返回搬完第一个 val 的迭代器；

然后套一个 erase，将这部分删除；

- 去掉首尾空格
```C++
int l = 0, r = s.size() - 1;
while (l <= r && s[l] == ' ') l++;
while (l <= r && s[r] == ' ') r--;
s = s.substr(l, r - l + 1);
```

## 大小写转换
```C++
#include <cctype>
for (char& c : s) {
    c = toupper(c); // 转大写，小写用 c = tolower(c)
}
```

## 字符串与数字转换

以下函数都在 std 命名空间中

- string -> int

```C++
int x = stoi(s);
```

- string -> long long

```C++
long long x = stoll(s);
```

- string -> double

```C++
double x = stod(s);
```

- int -> string

注意：各种数据类型转换成 `string` 的方法名是 `std::to_string`，而不是 `itos` 这种。

```C++
string s = to_string(x);
```

## 字符统计
- 统计字符出现次数
```C++
vector<int> cnt(26);
for (char c : s) {
    cnt[c - 'a']++;
}
```
- 判断是否是字母或数字（`std`）
```C++
isalpha(c);
isdigit(c);
isalnum(c);
```

## 常用 string API
```C++
s.size();

s.substr(int pos, int len);

// s.find() 返回值是 int，如果找不到，返回 string::npos
s.find(string str); // 寻找子串 str 第一次出现的位置，字符也是可以的
s.find(string str, size_t begin); // 从 begin 开始向后找
// 如：
string s = "ababab";
size_t pos = s.find("ab");
if (pos != string::npos) {
    cout << pos << endl;
    pos = s.find("ab", pos + 1);
}

// 删除
s.erase(int pos, int len);
s.earse(int pos); // 从 pos 删除到结尾
s.earse(iterator pos); // 按迭代器删除一个字符，如 s.earse(s.begin() + 2)
s.earse(iterator first, iterator last); // 按迭代器删除区间 [first, last)
// 插入
s.insert(int pos, int str);
s.insert(int pos, int cnt, char c); // 在 pos 插入 cnt 次 c
s.insert(iterator pos, char c); // 在 pos 插入一个字符 c
s.insert(iterator pos, int cnt, char c); // 在 pos 插入 cnt 次 c
// 插入另一个字符串的一个区间
s.insert(iterator pos, iterator first, iterator last); // 在 pos 插入另一个字符串的 [first, last) 如 s.insert(s.begin() + 1, t.begin(), t.end())
// 反转
reverse(s.begin(), s.end());
```
