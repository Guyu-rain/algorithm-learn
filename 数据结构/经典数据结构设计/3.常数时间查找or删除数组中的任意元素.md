# 常数时间删除/查找数组中的任意元素

这里用到的数据结构叫**哈希数组**。他由一个数组和哈希表组成。其中、`hashMap`是`valToIdx`，`array`是`idxToVal`。

其实一个`hashset`就可以随机查找、删除对应元素，为什么还要一个`array`？其实这类问题需要**随机**选出一个元素，既然要随机，那么这些元素就必须**稠密**。这是 hashSet 做不到的。

`array`如何在`O(1)`内删除一个元素？其实是通过与尾部元素进行交换，然后删除尾部元素来做到的。避免了对右侧元素的移动。

> 如果用`vector`作为 array，那么可以不用单独记录数组长度。

与随机读取元素有关的两个问题，如下：

## [380.常数时间插入、删除和获取随机元素](https://leetcode.cn/problems/insert-delete-getrandom-o1/)

本题最重要的是，要实现常数时间的插入、删除、获取随机元素。

其实，前两个要求就是 hashmap，通过一个哈希表就能够做到。但最后一个获取随机元素，由于 hashmap是稀疏的，没有办法直接索引，没有办法做到“等概率”随机获取元素。

除了 hashset，还有一些类似的数据结构，比如哈希链表(LinkedHashMap)，也就是主体是一个链表，同时用一个 hashMap 索引链表的 key。但链表这种数据结构也没有办法随机访问，没法做到常数时间获取随机元素。

要常数时间获取随机元素，底层必须是数组，也就是紧凑的数据结构。这样才能够随机访存。

因此，我们需要将 values 存储在一个 array 中。方便随机获取一个元素。同时，为了能够常数时间插入、删除：

1. 常数时间插入很简单，直接 push_back 即可；
2. 常数时间删除，需要一个 val->index 的映射，也就是通过一个 hashmap，能够快速找到元素在 array 中的位置。

那，array能够在 $O(1)$ 的时间中删除一个元素吗？可以！我们将要删除的元素和最后一个元素交换（这个过程同时需要更新 val->index 的映射），然后删除最后一个元素即可。这就是一个 $O(1)$ 的操作。

#### 注意：涉及到删除、插入操作的，需要先插后删

这里，我们交换要删除的元素到末尾，同时需要更新 `valToIndex`的映射，这两个操作都必须先改动、再删除。如果反过来，可能会将刚删除的又插入回来：

```C++
// 删除某个元素 val
int index = valToIndex[val];
int lastElem = nums.back();

// nums 先改后删
nums[index] = lastElem;
nums.pop_back();

// valToIndex 先改后删
valToIndex[lastElem] = index;
valToIndex.erase(val);
```

```C++
class RandomizedSet {
    // 存储元素的值
    private:
        vector<int> nums;
        // 记录每个元素对应在 nums 中的索引
        unordered_map<int, int> valToIndex;
        default_random_engine rand;
    
    public:
        RandomizedSet() {
            rand.seed(random_device()());
        }

        bool insert(int val) {
            // 若 val 已存在，不用再插入
            if (valToIndex.count(val)) {
                return false;
            }
            // 若 val 不存在，插入到 nums 尾部，
            // 并记录 val 对应的索引值
            valToIndex[val] = nums.size();
            nums.push_back(val);
            return true;
        }

        bool remove(int val) {
            // 若 val 不存在，不用再删除
            if (!valToIndex.count(val)) {
                return false;
            }
            // 先拿到 val 的索引
            int index = valToIndex[val];
            int lastElement = nums.back();
            // 交换 val 和最后一个元素
            nums[index] = lastElement;
            // 在数组中删除元素 val
            nums.pop_back();
            // 将最后一个元素对应的索引修改为 index
            valToIndex[lastElement] = index;
            // 删除元素 val 对应的索引
            valToIndex.erase(val);
            return true;
        }

        int getRandom() {
            // 随机获取 nums 中的一个元素
            uniform_int_distribution<int> dist(0, nums.size() - 1);
            return nums[dist(rand)];
        }
};
```

这里回顾一下随机数是怎么生成的；C++中的随机数是 `engine -> distribution` 的设计。

```C++
#include <random>

// 首先定义一个伪随机数生成器
default_random_engine e;

// 用真随机数给引擎播种
e.seed(device_random()());

// 声明一个均匀分布的 **分布对象**，范围是 [0, n]
uniform_int_distribution<int> dist(0, n);
// 分布对象负责将生成的随机数转换到规定的范围内
int x = dist(e); // 让引擎提供随机数，分布将其转换成 [0, n] 内的整数
```

## [710.黑名单中的随机数](https://leetcode.cn/problems/random-pick-with-blacklist/)

最后有效列表的总个数为 `sz = N - blacklist.size();`

总体思路还是差不多，由于 val 就是 1~sz 中的连续数字，因此不需要一个 array 额外存储。

定义一个 mapping，含义是对于在 `0~sz-1` 中的 val，如果在 blacklist 中，我们需要将其映射到`sz~N`中有效的 val。我们需要在构造函数中定义这个mapping：

- 如果 b 已经在区间 `[sz, N)` 内，直接忽略；
- 对于 `[0, sz - 1]` 中需要映射的（在 blacklist 中）元素 val，需要跳过 `[sz, N)` 中所有在 blacklist 中的。也就是说，如果遇到了 blacklist 中的元素，需要跳过，找一个不是的进行映射。如下：

```C++
class Solution {
    int sz;
    unordered_map<int, int> mapping;
    default_random_engine rand;
public:
    Solution(int N, vector<int>& blacklist) {
        sz = N - blacklist.size();
        // 这里就是创建了一个以当前时间为种子的随机数对象
        rand = default_random_engine(chrono::system_clock::now().time_since_epoch().count());

        // 先将 blacklist 转换成 set，方便查询
        unordered_map<int> blackSet;
        for (int& b : blacklist) {
            blackSet.insert(b);
        }

        // 将所有 [0, sz) 中，是黑名单中的元素映射为 [sz, N) 中不是黑名单的元素
        last = N - 1;
        // 跳过所有黑名单中的数字
        for (int& b : blacklist) {
            // 如果 b 在区间 [sz, N)，直接忽略
            if (b >= sz) continue;
            // 如果 b 在区间 [0, sz) 中，需要映射出去
            // 此时，需要确保映射的目标 last 不在黑名单中
            while (blackSet.contains(last)) {
                last--;
            }
            mapping[b] = last;
            last--;
        }
    }

    int pick() {
        // 随机选一个索引
        uniform_int_distribution<int> u(0, sz - 1);
        int index = u(rand);
        
        // 如果选中的 index 需要映射
        if (mapping.contains(index)) {
            index = mapping[index];
        }
        return index;
    }
};
```
