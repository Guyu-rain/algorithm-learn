# 线段树习题

在刷题之前需要熟悉线段树的种类。对于动态开点懒更新线段树，一共有两种：

- 赋值线段树；
- 区间累加线段树。

这两种线段树的主要区别是下传操作不同，对于不同的聚合方式：

- 赋值线段树下传时，直接将`lazyAssign`赋值给子节点的`lazyAssign`，将`hasLazy`改成`true`，然后依照聚合方式更新 `mergeVal`；
- 区间累加线段树下传时，需要将`lazyAssign`**累加**给子节点的`lazyAssign`，然后依照聚合方式更新`mergeVal`。

注意：聚合操作 `merger` 主要影响两个主要操作`update`，`query`

在做题之前，需要首先明确用到的**线段树种类、以及区间聚合方式**，然后再手撕一颗对应的线段树。判断线段树类型极其重要。看到区间操作题，如果是一个动态的数组，那么就要用线段树。而且肯定是**区间动态开点懒更新线段树**。如果判断错误树的类型，那么整个题目思路也会受阻。

换句话说，如果发现当前思路不通，返回来换一种线段树类型、然后想清楚区间聚合操作究竟是什么，一般都能够解出来。

小技巧💡：区间聚合操作有时并不好直接看出，但区间查询和区间更新句聚合操作相同，我们可以看区间查询的操作是什么。这个操作一般比较容易看出。

## 习题

### [729.我的日程安排表I](https://leetcode.cn/problems/my-calendar-i/)

实际上就是要维护一个区间，要高效更新、查询这个区间的值（本题为2值）。由于比较基础，因此可以手撕一颗**聚合关系为或（有值为1则为1），区间操作为赋值**的懒更新线段树。

这棵树实现起来并不难，但是要注意，调用`api`时，由于是右开区间，因此所有`endTime`要写成`endTime - 1`。

同时，我们可以用二叉搜索树来解决：什么情况下一个日程会有冲突？

1. 之前的日程还没有结束，这个日程就开始了；
2. 这个日程还没结束时，下个日程就开始了。

这样，我们可以用一个红黑树(`treemap`)存储每个日程。红黑树的 `lower_bound`和`upper_bound`都是 $O(logN)$ 的时间复杂度，其时间复杂度与线段树是差不多的。

```C++
class MyCalendar {
    // 记录日程，key 是开始时间，value 是结束时间
    // 这个和 set<pair<int, int>> 是等价的
    map<int, int> calendar;
public:
    MyCalendar() {}

    bool book(int start, int end) {
        // 找后一个活动：earlier.start >= this.start
        auto earlier = calendar.lower_bound(start);
        // 看看能否找到上界，如果能找到，同时比较结束时间
        if (earlier != calendar.end() && earlier->first < end) {
            // 本项目还没结束，下一个项目就开始了
            return false;
        }

        // 找前一个活动：later.start <= this.start
        auto later = calendar.upper_bound(start);
        if (later != calendar.begin() && start < (--later)->second) {
            // 前一个项目还没有结束，本项目就开始了
            return false;
        }
        // 安排日程
        calendar[start] = end;
        return true;
    }
};
```

这个方法主要是熟悉怎么找“前一个”、“后一个”项目。这种写法还是很模板的。

不过“日程表”这类题目如果问的稍微深一点，就是线段树的模板题了。所以还是要熟悉线段树，尤其是`动态实现 + 懒更新的线段树`。

### [699.掉落的方块](https://leetcode.cn/problems/falling-squares/)

区间操作，很容易想到用前缀和或线段树。但数组不停在变化，我们要记录每一个状态的最大值，所以只能用线段树。

我们考虑需要用什么线段树：

是累加线段树吗？如果按照这个想法，比如我们在区间 `[1, 9]` 上已经有一个方块，我们要在区间 `[5, 13]` 上累加，该怎么累加？好像是要先求区间的最大值？

其实，聚合操作很显然，就是`max`，我们每次查询就是查询最高的高度。但累加线段树好像行不通，这时想能不能用**赋值线段树**：给定一个区间，我们在区间上查询最大值，然后将这个最大值 + 方块高度赋值给这一段，是可行的。

因此：本题需要实现的是聚合操作为`max`的赋值线段树。如下：

```C++
class SegmentNode {
public:
    int l, r;
    int max;
    SegmentNode* left;
    SegmentNode* right;

    bool hasLazy;
    int lazyVal;

    SegmentNode(int l, int r, int max)
        : l(l), r(r), max(max), left(nullptr), right(nullptr), hasLazy(false), lazyVal(0) {}
};

// 聚合操作为最大值的赋值线段树
class AssignSegmentTree {
private:
    SegmentNode* root;

    void initChildrenIfNeeded(SegmentNode* node) {
        if (node->l == node->r) {
            return;
        }
        if (node->left == nullptr || node->right == nullptr) {
            int mid = node->l + (node->r - node->l) / 2;
            if (node->left == nullptr) {
                node->left = new SegmentNode(node->l, mid, 0);
            }

            if (node->right == nullptr) {
                node->right = new SegmentNode(mid + 1, node->r, 0);
            }
        }
    }

    void pushDown(SegmentNode* node) {
        if (!node->hasLazy) {
            return;
        }

        // 需要 pushDown，赋值线段树
        node->left->max = node->lazyVal;
        node->left->hasLazy = true;
        node->left->lazyVal = node->lazyVal;

        node->right->max = node->lazyVal;
        node->right->hasLazy = true;
        node->right->lazyVal = node->lazyVal;

        node->hasLazy = false;
    }

    // 赋值线段树
    void _updateRange(SegmentNode* node, int qL, int qR, int value) {
        // 完全覆盖
        if (qL <= node->l && node->r <= qR) {
            // 更新
            node->max = value;
            node->hasLazy = true;
            node->lazyVal = value;
            return;
        }

        initChildrenIfNeeded(node);
        pushDown(node);

        // 左右去找区间 
        int mid = node->l + (node->r - node->l) / 2;

        if (qR <= mid) {
            _updateRange(node->left, qL, qR, value);
        } else if (qL > mid) {
            _updateRange(node->right, qL, qR, value);
        } else {
            _updateRange(node->left, qL, mid, value);
            _updateRange(node->right, mid + 1, qR, value);
        }
        node->max = max(node->left->max, node->right->max);
    }

    int _query(SegmentNode* node, int qL, int qR) {
        if (qL <= node->l && node->r <= qR) {
            return node->max;
        }

        initChildrenIfNeeded(node);
        pushDown(node);

        int mid = node->l + (node->r - node->l) / 2;

        if (qR <= mid) {
            return _query(node->left, qL, qR);
        } else if (qL > mid) {
            return _query(node->right, qL, qR);
        } else {
            return max(_query(node->left, qL, mid), _query(node->right, mid + 1, qR));
        }
    }
// 打包几个接口
public:
    AssignSegmentTree(int start, int end) {
        // defaultVal = 0
        root = new SegmentNode(start, end, 0);
    }

    void rangeUpdate(int qL, int qR, int value) {
        _rangeUpdate(root, qL, qR, value);
    }

    int query(int qL, int qR) {
        return _query(root, qL, qR);
    }
};
```

然后对于每一个区间，求最大值、然后加上`lengthSize`赋值给这一区间即可。
