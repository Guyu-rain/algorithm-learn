### [查找和最小的 K 对数字](https://leetcode.cn/problems/find-k-pairs-with-smallest-sums/)

这个题实际上就是合并 K 个升序链表的变体。我们如何看待这两个数组？

例如：

```python
nums1 = [1, 7, 11]
nums2 = [2, 4, 6]
```

所有能够组成的数组可以组成三个链表，如下：

```python
[1, 2] -> [1, 4] -> [1, 6]
[7, 2] -> [7, 4] -> [7, 6]
[11, 2] -> [11, 4] -> [11, 6]
```

这里的三个“链表”是递增的，我们要做的，就是构造一个小根堆，然后按照可能的最小值、找出最小的`k`个数对即可。

```C++
vector<vector<int>> kSmallestPairs(vector<int>& nums1, vector<int>& nums2, int k) {
    auto cmp = [](const vector<int>& a, const vector<int>& b) {
        return (a[0] + a[1]) > (b[0] + b[1]);
    };
    priority_queue<vector<int>, vector<vector<int>>, decltype(cmp)> pq(cmp);

    for (int i = 0; i < nums1.size(); i++) {
        pq.push({nums1[i], nums2[0], 0});
    }

    vector<vector<int>> res;

    while (!pq.empty() && k > 0) {
        vector<int> cur = pq.top(); pq.pop();
        k--;

        int next_index = cur[2] + 1;
        if (next_index < nums.size()) {
            pq.push({cur[0], nums2[next_index], next_index});
        }

        res.push_back({cur[0], cur[1]});
    }
    return res;
}
```

## 寻找第 `K` 大元素

### [215.数组中的第K个最大元素](https://leetcode.cn/problems/kth-largest-element-in-an-array/)

最容易的想法，是维护一个大小为 K 的小根堆，当超过 K 时，弹出堆顶，等所有元素全部扫过一次之后，堆顶就是第`K`大元素。这种算法最为直观，事件复杂度为 $O(NlogK)$。

```C++
int findKthLargest(vector<int>& nums, int k) {
    priority_queue<int, vector<int>, greater<int>> pq;
    for (int e : nums) {
        pq.push(e);
        if (pq.size() > k) {
            pq.pop();
        }
    }
    return pq.top();
}
```

但其实最优的实现方式是快速选择算法。快速回顾一下，快速排序的本质，是二叉树的前序遍历（先选择出一个元素，通过调整两侧元素将该位置左和右分别有序）。快速选择如下：

```C++
default_random_engine e(random_device()());

int quickSelect(vector<int>& nums, int k) {
    int lo = 0, hi = nums.size() - 1;

    while (lo <= hi) {
        // 严格定义区间：
        // [lo, p] <= pivot，(p, hi] > pivot
        int p = partition(nums, lo, hi);
        if (p > k) {
            // k 不在 [p, hi] 中
            hi = p - 1;
        } else if (p < k) {
            // k 不在 [lo, p] 中
            lo = p + 1;
        } else {
            return nums[p];
        }
    }
    return -1; // 没有找到
}

int partition(vector<int>& nums, int lo, int hi) {
    uniform_int_distribution<int> u(lo, hi);
    
    int pivotIdx = u(e);
    int pivot = nums[pivotIdx];
    // 将 pivot 交换到第一个
    swap(nums[lo], nums[pivotIdx]);
    
    int i = lo + 1, j = hi;
    while (i <= j) {
        while (i <= hi && nums[i] <= pivot) {
            i++;
        }
        while (j >= lo && nums[j] > pivot) {
            j--;
        }
        if (i >= j) {
            break;
        }

        swap(nums[i], nums[j]);
    }

    swap(nums[lo], nums[j]);
    return j;
}
```

### [451.根据字符出现频率排序](https://leetcode.cn/problems/sort-characters-by-frequency/)

通过一个 `unordered_map<char, int>` 统计每一个字符出现的频率，然后将每一个这样的键值对`entry`压入大根堆中，然后弹出即可。

```C++
string frequencySort(string s) {
    vector<char> chars(s.begin(), s.end());

    unordered_map<char, int> charToFreq;
    for (char ch : chars) {
        charToFreq[ch]++;
    }

    // 设计一个大根堆，按照第二个元素大小进行排序
    auto cmp = [](const auto& a, const auto& b) {
        return a.second < b.second;
    }
    priority_queue<pair<char, int>, vector<pair<char, int>>, decltype(cmp)> pq(cmp);

    // 入堆
    for (const auto& entry : charToFreq) {
        pq.emplace(entry.first, entry.second);
    }

    string sb;
    while (!pq.empty()) {
        auto entry = pq.top(); pq.pop();
        sb.append(entry.second, entry.first);
    }

    return sb;
}
```

### [347.前K个高频元素](https://leetcode.cn/problems/top-k-frequent-elements/)

用一个 `hashMap valToFreq` 将每个元素的频率先提取出来。

然后，这个题就变成了 “找数组中第 K 个最大元素” 这个问题了。可以用大根堆来解决，或者用快速排序来解决。

有一种计数排序的思路：用一个数组`freqToVals`记录某个频率有哪些元素。这个映射的好处，是其自然的将频率进行了排序，然后倒序遍历这个数组即可。如下：

```C++
vector<int> topKFrequent(vector<int>& nums, int k) {
    // 统计每一个元素的频率
    unordered_map<int, int> valToFreq;
    for (const int& num : nums) {
        valToFreq[num]++;
    }

    // 用一个逆映射，freq -> val
    // freq 最大为 nums.size()，这里开一个大小为 nums.size() + 1 的数组，index范围是 [0, nums.size()]
    vector<int> freqToVals(nums.size() + 1);
    for (auto& entry : valToFreq) {
        int freq = entry.second;
        freqToVals[freq].push_back(entry.first);
    }

    // 记录前 k 个高频元素
    vector<int> res(k);
    int p = 0;
    // 到序遍历 freqToVals，得到的就是从大到小
    for (int i = freqToVals.size() - 1; i > 0; i--) {
        for (int val : freqToVals[i]) {
            // 将出现次数最多的 k 个元素装入 res
            res[p] = val;
            p++;
            if (p == k) {
                return res;
            }
        }
    }
    return {};
}
```

## 用两个二叉堆实现中位数算法

### [295.数据流的中位数](https://leetcode.cn/problems/find-median-from-data-stream/)

由于“树”各种相关的容器并没有提供随机访问迭代器，因此直接去找中间元素较为困难。

我们进行这样设计：设计**一个大根堆(small)，一个小根堆(large)**：

- 大根堆中的所有元素都要比小根堆小，因此大根堆叫`small`，小根堆叫`large`
- 两个堆元素之差为 `1`，也就是说两个堆的要么相同、要么有一个比另外一个大1。

此时，如果两个堆元素相同，中位数自然是两个堆堆顶的平均数，如果有一个堆元素比另一个多`1`，那么中位数就是这个元素更多的堆堆堆顶。

为了维护上述两个性质，我们先尝试维护第二个元素个数的性质：例如`small.size() >= large.size()`，那么就要像`large`中插入元素。但如果直接向插入`large`中插入元素，如果这个元素比`small`的堆顶要小，那么这个插入就是非法的，我们要保证`small`的堆顶永远小于`large`的堆顶。

但：如果我们想向`large`中插入元素，此时我们先向`small`中插入元素，此时，`small`的堆顶永远是`small`中最大的，将这个元素插入`large`，就能够较为精巧得维护这两条性质。

总之：想要往 `large` 中添加元素，必须先往 small 中添加，然后再将 `small` 堆顶的元素加到 `large` 中；向 `small` 中添加元素同理。

我们只需要维护着这两个性质，就能够轻易找到中位数。如下：

```C++
class MedianFinder {
    priority_queue<int> small; // 大根堆，记录小元素
    priority_queue<int, vector<int>, greater<int>> large; // 小根堆，记录大元素
public:
    MedianFinder() {}

    void addNum(int num) {
        // 添加时需要维护两个性质：
        // 1. small 的堆顶要比 large 小
        // 2. 两个堆的元素个数之差最多为 1
        if (small.size() <= large.size()) {
            // 最终目标：向 small 中插入元素
            large.push(num);
            small.push(large.top());
            large.pop();
        } else {
            // 最终目标：向 large 中插入元素
            small.push(num);
            large.push(small.top());
            small.pop();
        }
    }

    double findMedian() {
        if (small.size() < large.size()) {
            return large.top();
        } else if (small.size() > large.size()) {
            return small.top();
        } else {
            return static_cast<double>(large.top() + small.top()) / 2.0;
        }
    }
};
```
