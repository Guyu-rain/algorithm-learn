# 双指针解决数组问题
当`数组`本身有序，或者需要顺序处理时，需要考虑双指针。
双指针包含两种：
1. 快慢指针，经常解决原地操作的问题
2. 左右指针，如果数组有序，或者需要按顺序遍历，经常使用左右指针

## 双指针解决数组相关问题
### 1. 快慢指针技巧解决数组问题
#### 原地操作
##### 原地修改
使用快慢指针`fast`, `slow`。`fast`在前面“探路”，当`fast`找到一个新元素，那么就赋值给`nums[slow]`并让slow++。如下：
```C++
int removeDuplicates(vector<int>& nums) {
    // 特殊情况：空数组
    if (nums.size() == 0) { return 0; }
    int slow = 0, fast = 0;
    while (fast < nums.size()) {
        if (nums[slow] != nums[fast]) {
            nums[++slow] = nums[fast];
        }
        fast++; // fast 每一次循环都会自增
    }
    // slow 是索引，size 需要+1
    return slow + 1;
}
```

##### 原地删除
还是使用快慢指针，`fast`在前面探路，`slow`在后面指出应该放置的位置。如下：
```C++
int removeElement(vector<int>& nums, int val) {
    if (nums.empty()) { return 0; }
    int fast = 0, slow = 0;
    while(fast < nums.size()) {
        if (nums[fast] != val) {
            nums[slow++] = nums[fast];
        }
        fast++;
    }
    return slow;
}
```

##### 原地修改
也是使用快慢指针，此处不再重复。

### 2. 左右指针解决数组问题
#### 二分查找
```C++
int binarySearch(vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;
    while (left <= right) {
        int mid = (right + left) / 2;
        if (nums[mid] == target) {
            return mid;
        } else (nums[mid] < target) {
            left = mid + 1; // 注意 +1
        } else {
            right = mid - 1; // 注意 -1
        }
    }
    return -1;
}
```
#### `n`数之和(LeetCode167)
要找到两个数之和 `= target`，本质上是找到一种最快的遍历方式，在`n(n-1)/2`种可能性中找到两个要的数。
要让遍历更快，需要充分利用信息。已知：原数组有序，那么我们最好能够找到一种方法，能够数组中每个位置只遍历一次，就能够找到这两个数：让`left`, `right`分别指向最小、最大两个数，让这两个数相加，若比`target`小，那么`left++`，若比`target`大，那么`right--`，如下。
```C++
pair<int, int> twoSum(vector<int>& numbers, int target) {
    int left = 0, right = numbers.size() - 1;
    // 注意结束的判断条件，left == right已经不符合题意了
    while (left < right) {
        int sum = numbers[left] + numbers[right];
        if (sum == target) {
            // 最好指定一下返回类型
            // 题目要求索引从 1 开始
            return pair<int, int>{left + 1, right + 1}; 
        } else (sum < target) {
            // 当前值太小，left++;
            left++;
        } else if (sum > target) {
            // 当前值太大，right--;
            right--;
        }
    }
    return pair<int, int>{-1, -1}
}
```

#### 反转数组
基本的反转数组应该都能想到用左右指针
```C++
void reverseString(vector<char>& s) {
    int left = 0, right = s.size() - 1;
    while (left < right) {
        std::swap(s[left--], s[right++]);
    }
}
```

#### 回文数判断(LeetCode 5)
找以某个中心的最长回文数/串，由于回文串长度可能是单数或者双数，因此我们考虑以`s[l], s[r]`为中心的最长回文子串如下：
```C++
string getMaxPalindrome(string s, int l, int r) {
    // 注意防止索引越界
    while (l >= 0 && r < s.size() 
            && s[l] == s[r]) {
        l--; r++;
    }
    // 此时的最长回文子串就是 s[l+1..r-1]
    return s.substr(l+1, r - l - 1);
}
```
为了找最长的回文子串，我们还是需要遍历所有的位置，只不过需要注意回文子串的中心可能是`1`或者`2`（回文子串长度是奇数/偶数）：
```
for 0<= i < len(s):
    找到以 s[i] 为中心的子串
    找到以 s[i+1] 为中心的子串
    更新答案
```
因此，找最长回文子串如下：
```C++
string longestPalindrome(string s) {
    string res = "";
    for (int i = 0; i < s.length; i++) {
        // 调用上面那个函数
        string s1 = getMaxPalindrome(s, i, i);
        string s2 = getMaxPalindrome(s, i, i + 1);
        // res = longest(res, s1, s2);
        res = res.length() > s1.length() ? res : s1;
        res = res.length() > s2.length() ? res : s2;
    }
    return res;
}
```
