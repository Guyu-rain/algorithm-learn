# 二叉树算法模板
## 两种思想
所有二叉树问题都可以用两种思想解决：
1. 遍历问题：是否可以通过**遍历**一遍得到答案 —— 回溯算法
2. 分解问题：是否可以定义一个**递归**函数，通过解决子问题（子树）的答案推导出原问题的答案 —— 动态规划

一般而言：
- 通过遍历解决问题时函数签名通常是`void traverse(...)`，没有返回值，通过更新外部变量计算结果。
- 分解问题需要根据函数具体功能而定，而且一般会有返回值，这个返回值就是子问题的结果。一般是后序位置进行处理。
换言之，一旦题目和子树有关，那么大概率需要合适的函数返回值，在后序位置写代码（见下文**后序位置的特殊之处**）

### 前中后序遍历
```C++
void traverse(TreeNode* root) {
    if (root == nullptr) { 
        return;
    }

    // 前序位置
    traverse(root->left);
    // 中序位置
    traverse(root->right);
    // 后序位置
}
```
前中后序其实并没有本质的区别，只不过是**什么时机“访问”**一个节点。
二叉树的所有问题，就是在前中后序位置加入代码逻辑，从而达到目的。我们只需要**单独思考**每个节点应该做什么，其余的事情不用管，二叉树遍历框架会在虽有节点上递归做相同操作。

### 两种思想的变通性
二叉树的所有问题，本质上还是对树进行遍历，有三种遍历方式。所以其实两种算法是可以变通的，可以用两种思想找到问题的解。
但是，在实施的过程中代码细节、思维难度可能是不同的，因此需要挑选一种自然的方法解决。

### 后序位置的特殊之处
- 前序遍历没有什么特别之处，我们习惯将对位置不敏感的问题，每个节点的处理放在前序位置。也就是第一次进入一个节点时就进行处理。
- 中序位置主要用在 BST 中，用中序遍历扫一遍 BST 能够得到有序的结果
> 注意：前序->中序->后序 位置的代码，能力依次增强
> 前序位置代码只能从**函数参数**中获取**父节点**传来的数据
> 中序位置不仅可以获取参数（父节点）数据，还可以获取到左子树通过函数返回值传递回来的数据
> 后序位置的代码能力最强，不仅可以获取参数数据，还可以同时获得左右子树传递回来的数据。
因此：
- 有些情况下代码移到后序效率最高，但有些问题，只能在后续位置的代码做。
- 一般涉及到子树的问题，都需要后序遍历来进行，只有在后序能够拿到左右子树的信息；同时，一般还需要对函数设计一个返回值，也就是子树的相关信息。

### 动态规划/回溯/DFS算法的区别与联系
动态规划、DFS、回溯都是二叉树的扩展，只不过其关注点不同：
1. 动态规划数据**分解（分治）**的思路，关注点在整颗**子树**。
    - 如：计算树一共有多少节点，递归斐波那契数列。
2. 回溯算法属于**遍历**的思路，关注点在节点间的**树枝**。
    - 如：打印遍历二叉树的过程（见下方代码）
3. DFS算法属于**遍历**的思路，关注点在单个**节点**。
    - 将二叉树上每个节点值 +1
用回溯算法打印多叉树遍历的过程：
```C++
class Node {
    int val;
    Node[] children;
};

void traverse(Node root) {
    if (root == null) return;
    for (Node child : root.children) {
        printf("从节点 %s 进入节点 %s", root, child);
        traverse(child);
        printf("从节点 %s 回到节点 %s", child, root);
    }
}
```
注意，DFS和回溯算法区别主要在：
- DFS “做选择”，“撤销选择” 在 for 循环**外面**
    - 这是因为 DFS 只关心当前进入的节点
- 回溯算法 “做选择”，“撤销选择” 在 for 循环**里面**
    - 这是因为回溯算法需要关注树枝，需要拿到树枝的两端节点。
```C++
// dfs：在 for 循环外做选择，关注每个节点
void dfs(Node* root) {
    if (!root) return;

    // 做选择
    printf("enter node %s\n", root->val.c_str());
    for (Node* child : root->children) {
        dfs(child);
    }
    // 撤销选择
    printf("leave node %s\n", root->val.c_str());
}

// 回溯：在 for 循环中做选择，关注“从root到child进入、离开的过程”（树枝）
void backtrack(Node* root) {
    if (!root) return;
    for (Node* child : root->children) {
        // 做选择
        printf("I'm on the branch from %s to %s\n", root->val.c_str(), child->val.c_str());
        backtrack(child);
        // 撤销选择
        printf("I'll leave the branch from %s to %s\n", child->val.c_str(), root->val.c_str());
    }
}
```

### 层序遍历
主要用来求最短路径问题
```C++
int levelTraverse(TreeNode* root) {
    if (root == nullptr) return;
    
    queue<TreeNode*> q;
    q.push(root);

    int depth = 0;
    while (!q.empty()) {
        int sz = q.size();
        for (int i = 0; i < sz; i++) {
            TreeNode* cur = q.front(); q.pop();

            // 将下一层入队
            if (cur->left != nullptr) {
                q.push(cur->left);
            }
            if (cur->right != nullptr) {
                q.push(cur->right);
            }
        }
        depth++;
    }
    return depth;
}
```
