# 快速排序

快速排序的本质是二叉树的前序遍历。选择一个数当`pivot`，然后将数组中小于等于`pivot`的元素放在`pivot`左，将大于`pivot`的放在右侧。然后左右递归处理整个数组。

快速排序需要解决两个问题：

1. 如果我们就选择 `nums[lo]` 作为 `pivot`，当数组完全有序时，此时每次递归都会扫描整个数组，时间复杂度退回$O(N^2)$
2. 如果有大量相同元素，相同的，也会退回到 $O(N^2)$

#### 其中，数组有序的问题可以通过 随机选择 pivot 或者 将原数组打乱 来解决

将原数组打乱，然后固定选择`pivot = nums[lo]`的方法如下：

```C++
// 过程：选择一个 pivot，将数组分成两部分
// 将所有比 pivot 小的放在 pivot 左边，比 pivot 大的放在 pivot 右边 —— 实际上就是将 pivot 排好序
// 递归排序左右两部分 - 二叉树的 DFS

// 有两个需要额外解决的问题
// 1. pivot 的选择需要随机
// 2. 初始顺序需要随机，尽量使树高为平衡二叉树；
// 否则算法最差可能退化到 O(N^2)

// 时间复杂度：O(NlogN)
// 原地排序，空间复杂度：O(logN) - 来自递归调用栈
// 是不稳定排序

class QuickSort {
public:
    // 使用 static 关键字确保无需实例化 QuickSort 即可调用
    static void sort(vector<int>& nums) {
        // 为了避免极端情况，先随机打乱
        shuffle(nums); 
        // 排序整个数组（原地修改）
        sort(nums, 0, nums.size() - 1); // 二叉树的遍历
    }
private:
    static void sort(vector<int>& nums, int lo, int hi) {
        if (lo >= hi) { return; }
        // 对 nums[lo..hi] 进行切分
        // 使 nums[lo..p-1] <= nums[p] < nums[p+1..hi]
        int p = partition(nums, lo, hi);
        
        sort(nums, lo, p - 1);
        sort(nums, p + 1, hi);
    }

    // 对 nums[lo..hi] 进行切分
    static int partition(vector<int>& nums, int lo, int hi) {
        int pivot = nums[lo];
        // 定义 i, j 使 nums[lo..hi] 满足：
        // [lo, i) <= pivot; (j, hi] > pivot
        int i = lo + 1, j = hi;
        
        while (i <= j) {
            while (i < hi && nums[i] <= pivot) { i++; }
            while (j > lo && nums[j] > pivot) { j--; }
            // 到这里，能保证[lo, i) <= pivot; (j, hi] > pivot
            // 换言之，如果 i, j 如果没有到边界，此时有：
            // nums[i] > pivot, nums[j] <= pivot

            // 所以应该交换这两个位置
            if (i >= j) { break; }
            swap (nums[i], nums[j]);
            // 此时，[lo, i] <= pivot && [j, hi] > pivot
        }
        // 此时，满足了[lo, i) <= pivot; (j, hi] > pivot
        // 特别的，j <= pivot，不会超出范围（pivot 在 lo，最多和自己交换）
        // 将 pivot 放在合适的位置
        swap(nums[lo], nums[j]);
        return j;
    }

    static void shuffle(vector<int>& nums) {
        srand((unsigned) time(NULL));
        int n = nums.size();
        for (int i = 0; i < n; i++) {
            int r = i + rand()%(n-i);
            swap(nums[i], nums[r]);
        }
    }
};
```

通过随机选择`pivot`，解决数组有序的方式如下：

```C++
default_random_engine e(random_device()());

void quickSort(vector<int>& nums, int lo, int hi) {
    // base case，没有元素，直接返回
    if (lo > hi) {
        return;
    }

    int p = partition(nums, lo, hi);

    sort(nums, lo, p - 1);
    sort(nums, p + 1, hi);
}

int partition(vector<int>& nums, int lo, int hi) {
    // 随机选择一个元素当作 pivot
    uniform_int_distibution<int> u(lo, hi);
    int pivotIdx = nums[u(e)];
    // 将 pivot 与 lo 调换，换到数组开始的位置
    swap(nums[lo], nums[pivotIdx]);
    int pivot = nums[pivotIdx];

    int i = lo + 1, j = hi;
    
    // 严格定义：i, j 为开区间 且：
    // [lo, i) <= pivot, (j, hi] > pivot
    while (i <= j) {
        while (i <= hi && nums[i] <= pivot) i++;
        // 此时，一定有 nums[i] > pivot（或i > hi）
        while (j >= lo && nums[j] > pivot) j--;
        // 此时，一定有 nums[j] <= pivot（或 j < lo）

        if (i >= j) break;
                
        swap(nums[i], nums[j]);
    }
    // 将 lo 交换回来
    swap(nums[lo], nums[j]);
    return j;
}
```

注意：需要考虑清楚`i == j`是什么情况。我们定义 `i = lo + 1，j = hi`。

1. 如果刚进入循环时，`i == j`，说明只有两个元素。这两个元素也需要排序，因此要准入，所以while循环条件必须写成 `i <= j`
> 比如数组`{3, 2}`，需要进行排序，所以要准入。

2. if 的 `break`。什么时候会出现 `i == j`？正常退出两个循环一定有 `nums[i] <= 
pivot`，`nums[j] > pivot`，因此肯定不是正常退出，检查一下是否会卡边界：也就是 `i == j == hi`。好像也不会，因此这里可以写成 `i > j` `break`。不过为了统一，以及更严格的边界，这里也用了 `i >= j` 进行统一。

#### 有大量重复元素时，可以通过三路快排解决

三路快排严格分成四个区域：

```
[lo, lt) < pivot
[lt, i) == pivot
[i, gt]             未检查
(gt, hi] > pivot
```

```C++
default_random_engine e;

void sort(vector<int>& nums, int lo, int hi) {
    if (lo >= hi) return;

    e.seed(random_device()());
    auto [lt, gt] = partition(nums, lo, hi);

    sort(nums, lo, lt - 1);
    sort(nums, gt + 1, hi);
}

pair<int, int> partition(vector<int>& nums, int lo, int hi) {
    uniform_int_distribution<int> u(lo, hi);
    int pivotIdx = u(e);
    swap(nums[lo], nums[pivotIdx]);
    int pivot = nums[lo];

    // 严格定义 lt, gt 为开区间，且：
    // [lo, lt) < pivot, [lt, gt] == pivot, (gt, hi] > pivot
    int lt = lo, gt = hi, i = lo + 1;

    while (i <= gt) {
        if (nums[i] < pivot) {
            swap(nums[lt], nums[i]);
            lt++; i++;
        } else if (nums[i] > pivot) {
            swap(nums[gt], nums[i]);
            gt--;
            // i 不前进，换来的元素还未检查
        } else {
            i++;
        }
    }

    return {lt, gt};
}
```
