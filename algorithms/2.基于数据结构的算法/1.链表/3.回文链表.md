# 回文链表
在数组中：
- **寻找**回文：通过中间向两端扩展
```C++
string palindrome(string s, int left, int right) {
    // 防止越界
    while (left >= 0 && right < s.length() && s[left] == s[right]) {
        left--;
        right++;
    }
    // 返回以 s[left], s[right] 为中心的最长子串
    return s.substr(left + 1, right - left - 1);
}
```
- **判断**回文：从两端向中间逼近
```C++
bool isPalindrome(string s) {
    int left = 0, right = s.length() - 1;
    while (left < right) {
        if (s[left] != s[right]) {
            return false;
        }
        left++;
        right--;
    }
    return true;
}
```

## 判断单链表是否回文[LeetCode234.回文链表](https://leetcode.cn/problems/palindrome-linked-list/)

> 反转相关，应该能够想到栈这种数据结构。
最简单的想法：先将链表反转(递归/迭代方式)，得到一条新的链表，然后比较这两条链表是否相同。

#### 链表具有递归结构，树也是链表结构的延伸
```C++
// 遍历单链表
void traverse(ListNode* head) {
    // base case
    // 前序遍历
    traverse(head->next);
    // 后序遍历
}
```
树是其延伸：
```C++
void traverse(TreeNode* root) {
    // base case
    // 前序遍历代码
    traverse(root->left);
    // 中序遍历代码
    traverse(root->right);
    // 后续遍历代码
}
```
可以通过后序遍历的想法，来解决这个问题：
```C++
ListNode* left;  // 从左向右 正向移动的指针
ListNode* right; // 从右向左 反向移动的指针
bool res = true;

bool isPalindrome(ListNode* head) {
    left = head;
    traverse(head);
    return res;
}

void traverse(ListNode* right) {
    if (head == nullptr) {
        return;
    }
    if (!res) {
        return;
    }
    // 先一路走到链表尾部
    traverse(right->next);
    if (right->val != left->val) {
        res = false;
    }
    left = left->next;
}
```
这个做法的本质：将链表放入一个栈，栈顶始终用`right`指着。然后不断`pop`，元素就自然是反的。这种做法时间、空间复杂度都是 `O(N)`。

### 优化空间复杂度
想到数组：我们能否将链表构造成可以从两侧向中间不断检测的形式？

如果我们找到链表中点（快慢指针）
```C++
1->2->3(slow)->4->5(fast)->nullptr
```
如果是偶数，则有类似：
```C++
1->2->3->4(slow)->5->6->nullptr(fast)
```
然后我们将这条链表从`slow`开始反转(reverse)，并且用left和right指向两端：
```C++
1(left)->2->3<-4<-5(right)
            ↓
          nullptr
```
如果是偶数，则有：
```C++
1(left)->2->3->4<-5<-6(right)
               ↓
            nullptr
```
此时，从两侧向中间查找，只要`right == nullptr`之前没有出现不同，就是回文串。有代码如下：
```C++
bool isPalindrome(ListNode* head) {
    ListNode *slow, *fast;
    slow = fast = head;
    while (fast != nullptr && fast->next != nullptr) {
        fast = fast->next->next;
        slow = slow->next;
    }
    // 反转链表
    ListNode* left = head;
    ListNode* right = reverse(slow);
    while (right != nullptr) {
        if (right->val != left->val) {
            return false;
        }
        right = right->next;
        left = left->next;
    }
    return true;
}

ListNode* reverse(ListNode* head) {
    ListNode *pre = nullptr, *cur = head;
    while (cur != nullptr) {
        // 可以在这里声明 nxt，这样可以省去 base case
        // head == nullptr 或者 head->next == nullptr 都是不会出错的
        // 向前移动时，不需要移动 nxt
        ListNode* nxt = cur->next;
        // 反转
        cur->next = pre;
        pre = cur;
        cur = nxt;
    }
    return pre;
}
```
如果想要保证遍历后数组有序，需要得到`p, q`两个位置，然后`p->next = reverse(q)`即可。
```C++
1->2->3(p)->4(slow)<-5<-6(q)
          ↓
        nullptr
```
其中，`q`的位置就是`right`起始位置。`p`的位置需要特殊记录，是`slow`的前一个位置，需要快慢链表阶段多一个位置来记录。
