# 双指针算法

## 基本知识见`1.基础基础算法 1.链表双指针算法部分`
当链表本身有序，或考察链表自身是否有环、是否相交，需要考虑双指针算法。

## 链表的分解
链表可以通过某些限定条件分解成多条链表，一般需要通过维护这些链表的虚拟头节点`dummy` + 用来遍历这些链表和原链表的指针`p_i`。

### [LeetCode82.删除排序链表中的重复元素](https://leetcode.cn/problems/remove-duplicates-from-sorted-list-ii/)
#### 思路1：考虑将链表分解成重复和不重复两条链表
使用双指针的方式，将链表分解成两条链表：
```C++
ListNode* deleteDuplicates(ListNode* head) {
    ListNode dummyUniq(-1);
    ListNode dummyDup(-1);

    ListNode* pUniq = &DummyUniq;
    ListNode* pDup = &dummyDup;
    ListNode* p = head;

    while (p != nullptr) {
        // 重复节点：p->next->val == p->val && p->next != nullptr
        // 或者直接与重复链表上的节点相同：p->val == pDup->val;
        if ((p->next->val == p->val && p->next != nullptr) || p->val == pDup->val) {
            pDup->next = p;
            pDup = pDup->next;
        } else {
            // 不是重复节点
            pUniq->next = p;
            pUniq = pUniq->next;
        }
        p = p->next;
        // 将原链表和新链表断开
        pUniq->next = nullptr;
        pDup->next = nullptr;
    }
    return dummyUniq.next;
}
```

#### 思路2：用快慢双指针
[LeetCode83.删除排序链表中的重复元素](https://leetcode.cn/problems/remove-duplicates-from-sorted-list/description/)很相似，只不过本题要求将重复的全部删除。
```C++
ListNode* deleteDuplicated(ListNode* head) {
    ListNode *slow, *fast; fast = head;
    ListNode dummy(101);
    slow = &dummy;
    int dup = 101;

    while (fast != nullptr) {
        // 如果出现重复，fast 也要移动到重复元素的最后一个上
        // 注意：可能会出现连续的重复元素，如：`1, 1, 2, 2, 3, 4, 4` 需要移动到 3 上

        // 出现了重复元素，开始向后找
        // 注意：如果是结尾有重复，那么会移动到最后一个元素上，因此下面要做结尾特判
        while ((fast->next != nullptr && fast->val == fast->next->val)) {
            fast = fast->next;
            dup = fast->val;
            // 如果此时已经找到“头”，向后一个
            if (fast->next != nullptr && fast->val != fast->next->val) {
                fast = fast->next;
                // 假设这个是不重复的
                dup = 101;
            }
        }
        // 结尾特判：看看结尾元素(fast->next == nullptr)能不能要(dup != 101)，不能要直接break；能要和下面逻辑合并起来
        if (fast->next == nullptr && dup != 101) {
            break;
        }

        slow->next = fast;
        fast = fast->next;
        slow = slow->next;

        slow->next = nullptr;
        dup = 101;
    }

    return dummy.next;
}
```
或
```C++
ListNode* deleteDuplicates(ListNode* head) {
    ListNode dummy(-1);
    ListNode* p = &dummy; // “慢指针”
    ListNode* q = head; // “快指针”
    while (q != nullptr) {
        if (q->next != nullptr && q->val == q->next->val) {
            // 发现重复节点，跳过这些节点
            while (q->next != nullptr && q->val == q->next->val) {
                q = q->next;
            }
            // 跳到不重复的元素上
            q = q->next;
            // 此时 q 跳过了这一段重复元素
            // 检查是否到了结尾，这里做收尾
            if (q == nullptr) {
                p->next = nullptr;
            }
        } else {
            // 不是重复节点，直接拼接上去
            p->next = q;
            p = p->next;
            q = q->next;
        }
    }
    return dummy.next;
}
```
### [LeetCode378.有序矩阵中第 K 小的元素](https://leetcode.cn/problems/kth-smallest-element-in-a-sorted-matrix/)
转化一下，立马变成[LeetCode23.合并K个升序链表](https://leetcode.cn/problems/merge-k-sorted-lists/description/)
```C++
int kthSmallest(vector<vector<int>>& matrix, int k) {
    int n = matrix.size();
    // 因为不是链表，需要存储是从哪个数组中选出来的
    // 用一个小根堆，存储tuple (matrix[i][j], i, j)
    
    priority_queue<tuple<int, int, int>, vector<tuple<int, int, int>>, greater<tuple<int, int, int>>> pq;

    // 初始化优先级队列，将第一行的每一个元素装入
    for (int i = 0; i < n; i++) {
        pq.push({matrix[i][0], i, 0});
    }

    int res = -1;
    // 找到第 k 小的元素
    while (k--) {
        tuple<int, int, int> cur = pq.top(); pq.pop();
        auto& [num, i, j] = cur;
        
        res = num;
        if (j + 1 < n) {
            pq.push({matrix[i][j], i, j + 1});
        }
    }
    return res;
}
```

## 链表运算
### [LeetCode2.两数相加](https://leetcode.cn/problems/add-two-numbers/)
没什么技术含量，注意用一个虚拟头节点 `dummy`，然后注意处理进位 `carry`
```C++
class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        // 在两条链表上的指针
        ListNode *p1 = l1, *p2 = l2;
        // 虚拟头结点（构建新链表时的常用技巧）
        ListNode *dummy = new ListNode(-1);
        // 指针 p 负责构建新链表
        ListNode *p = dummy;
        // 记录进位
        int carry = 0;
        // 开始执行加法，两条链表走完且没有进位时才能结束循环
        while (p1 != nullptr || p2 != nullptr || carry > 0) {
            // 先加上上次的进位
            int val = carry;
            if (p1 != nullptr) {
                val += p1->val;
                p1 = p1->next;
            }
            if (p2 != nullptr) {
                val += p2->val;
                p2 = p2->next;
            }
            // 处理进位情况
            carry = val / 10;
            val = val % 10;
            // 构建新节点
            p->next = new ListNode(val);
            p = p->next;
        }
        // 返回结果链表的头结点（去除虚拟头结点）
        return dummy->next;
    }
};
```

### [LeetCode445.两数相加II](https://leetcode.cn/problems/add-two-numbers-ii/)
注意：遇到“反转”就想**栈**这种数据结构。
- 判断好压栈的时机，出栈就自动变成我们需要的顺序。注意需要头插保证顺序。
```C++
#include <stack>

class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        // 把链表元素转入栈中
        stack<int> stk1;
        while (l1 != nullptr) {
            stk1.push(l1->val);
            l1 = l1->next;
        }
        stack<int> stk2;
        while (l2 != nullptr) {
            stk2.push(l2->val);
            l2 = l2->next;
        }
        
        // 接下来基本上是复用我在第 2 题的代码逻辑
        // 注意新节点要直接插入到 dummy 后面

        // 虚拟头结点（构建新链表时的常用技巧）
        ListNode* dummy = new ListNode(-1);

        // 记录进位
        int carry = 0;
        // 开始执行加法，两条链表走完且没有进位时才能结束循环
        while (!stk1.empty() || !stk2.empty() || carry > 0) {
            // 先加上上次的进位
            int val = carry;
            if (!stk1.empty()) {
                val += stk1.top();
                stk1.pop();
            }
            if (!stk2.empty()) {
                val += stk2.top();
                stk2.pop();
            }
            // 处理进位情况
            carry = val / 10;
            val = val % 10;
            // 构建新节点，直接接在 dummy 后面
            ListNode* newNode = new ListNode(val);
            newNode->next = dummy->next;
            dummy->next = newNode;
        }
        // 返回结果链表的头结点（去除虚拟头结点）
        return dummy->next;
    }
};
```

## 链表环检测
### [LeetCode287.寻找重复数](https://leetcode.cn/problems/find-the-duplicate-number/)
这个题目体现了内容和下标的转换，形象理解是**数值(value)、位置(index)**的转换。

这个题的约束很特别：数组长度`n+1`，元素数值$\in [1, n]$。需要一些经验，能够发现如果将每个位置作为一个状态，那么实际上整个过程描述了一个状态转移：`0 -> nums[0]->nums[nums[0]]->...` 不断向后。如果我们将这个过程视为一个链表：
- 如果没有重复数字，这个链表不会成环，是一个常规的单链表；
- 如果链表有重复数字`nums[i] == nums[j] == k`，说明有两个节点 `i, j` 指向了节点 `k`，两进一出：成环。
综上，题目转换成求单链表环起点的问题，这与[142.环形链表II](https://leetcode.cn/problems/linked-list-cycle-ii/description/)完全等价。
```C++
int findDuplicate(vector<int>& nums) {
    // 下方是单链表环检测算法，将数组当成一种链表
    int fast = 0, slow = 0;
    while (true) {
        fast = nums[nums[fast]];
        slow = nums[slow];

        if (fast == slow) { break; }
    }
    // 重新指向头节点（索引0）
    slow = 0;
    while (slow != fast) {
        fast = nums[fast];
        slow = nums[slow];
    }
    return slow;
}
```
