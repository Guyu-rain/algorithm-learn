# 二叉树心法

所有二叉树题目都需要分成两类：

1. 是否可以通过**遍历**一次得到答案？如果可以，用一个 `traverse` 函数配合外部变量来实现。这叫“遍历”的思维模式。
2. 是否可以定义一个**递归函数**，通过子问题（子树）的答案推导出原问题的答案？如果可以，写出这个函数的廷议，并充分利用这个函数的返回值，这叫“分解问题”的思维模式。

无论用哪种思维模式，都需要思考：

如果单独抽出一个二叉树节点，需要做什么事情？需要在什么时候（前/中/后）做？其他节点并不需要操心，递归函数会在所有节点上做相同的事。

## 思路篇
### [226.翻转二叉树](https://leetcode.cn/problems/invert-binary-tree/)

是否可以用遍历的思维模式解决？是可以的，让每个节点的左右子节点颠倒过来就可以了。

```C++
TreeNode* invertTree(TreeNode* root) {
    if (root != nullptr) traverse(root);
    return root;
}

void traverse(TreeNode* root) {
    if (root == nullptr) return;

    // 交换左右子树
    TreeNode* temp = root->left;
    root->left = root->right;
    root->right = temp;

    // 去所有子节点上做相同的事情
    traverse(root->left);
    traverse(root->right);
}
```

是否可以用分解问题的思维模式解决？一般用分解问题都是有返回值的，我们利用题目中的函数签名即可，定义`invertTree`函数：将以 root 为根的树翻转，返回反转后的二叉树跟节点。

```C++
TreeNode* invertTree(TreeNode* root) {
    if (root == nullptr) return root;
    // 将以 root 为根的子树翻转，并返回
    TreeNode* reverseLeft = invertTree(root->left);
    TreeNode* reverseRight = invertTree(root->right);

    // 交换左右子树
    root->left = right;
    root->right = left;

    return root;
}
```

### [116.填充每个二叉树节点的右指针](https://leetcode.cn/problems/populating-next-right-pointers-in-each-node/)

最朴素的想法，是遍历二叉树，将同一层的节点先按照顺序存储起来，但这会花额外的$O(N)$的空间复杂度。这显然不好。

能否用“遍历”的思维模式解决？如果我们用遍历的方式尝试解决，我们容易将同一个根的两个字节点之间用 `next` 指针连接起来，但左右子树之间的“空隙”却连接不起来。

这个题需要抽象这颗二叉树为三叉树，相邻的两个节点看成一个节点。这样就相当于每个根节点下有三个叶子节点。然后遍历这个三叉树，将每个“叶节点内部相连”即可。还是需要一些技巧的，看如下的代码吧：
```C++
Node* connect(Node* root) {
    if (root == nullptr) return root;
    // 遍历“三叉树”
    traverse(root->left, root->right);
    return root;
}

void traverse(Node* node1, Node* node2) {
    // 如果三叉树的“叶子节”点不“完整”，直接 return
    // 这时候没有需要连接的 next 指针
    if (node1 == nullptr || node2 == nullptr) {
        return;
    }
    node1->next = node2;
    // 遍历“三叉树”
    traverse(node1->left, node1->right);
    traverse(node1->right, node2->left);
    traverse(node2->left, node2->right);
}
```

这个题没有什么好的分解解法。

### [114.将二叉树展开为链表](https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/)

遍历能解决吗？遍历每个节点，然后通过前序的方式构建这一个链表：

```C++
TreeNode* dummy = new TreeNode(-1);
TreeNode* p = dummy;

void traverse(TreeNode* root) {
    if (root == nullptr) return;

    // 前序位置
    p->right = new TreeNode(root->val);
    p = p->next;

    traverse(root->left);
    traverse(root->right);
}
```

这样做的空间复杂度是$O(n)$; 而且原函数的签名是 `void`，希望我们原地改动。虽然可以容国 `root = dummy->right`这种方式强行改变root指向，但并不是最优。

尝试“分解”的思路：如果左右子树都“展平”了，那么我们要做的，就是将右子树拼接在左子树的最后一个节点后面。如下

```C++
void flatten(TreeNode* root) {
    if (root == nullptr) return;

    // 展平左子树
    flatten(root->left);
    // 展平右子树
    flatten(root->right);

    // 将展平的右子树拼接在展平的左子树后面
    // 如果左子树空，此时已经满足条件
    if (root->left == nullptr) return;
    // 左子树不空
    TreeNode* p = root->left;
    while (p->right != nullptr) {
        p = p->right;
    }
    
    p->right = root->right;
    root->right = root->left;
    root->left = nullptr;
}
```

## 构造篇

**二叉树的构造问题一般都是用“分解问题”的思路：构造整棵树 = 根节点 + 构造左子树 + 构造右子树。**

### [654.最大二叉树](https://leetcode.cn/problems/maximum-binary-tree/)

题目中已经给出了构造的方式。我们只需要模拟题目中要求即可。为了在一个数组上用指针在划定范围，我们重新创建一个函数`build`用来构造。

```C++
// 子数组范围：[lo, hi)
TreeNode* build(vector<int>& nums, int lo, int hi) {
    // 实际上就是 lo == hi
    if (lo >= hi) return nullptr;

    // 找到最大值
    int maxIdx = lo;
    for (int i = lo; i < hi; i++) {
        maxIdx = nums[maxIdx] > nums[i] ? maxIdx : i; 
    }
    // 构造二叉树：根 + 左 + 右
    TreeNode* root = new TreeNode(nums[maxIdx]);
    // 注意边界，左闭右开
    root->left = build(nums, lo, maxIdx);
    root->right = build(nums, maxIdx + 1, hi);

    return root;
}

TreeNode* constructMaximumBinaryTree(vector<int>& nums) {
    TreeNode* root = build(nums, 0, nums.size());
    return root;
}
```

### [105.通过前序与中序遍历构造二叉树](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/)

还是构造问题。方法肯定是**分解思路：通过`根 + 左子树 + 右子树`来解决**。

我们知道，前序的序列是“根 | 左子树前序 | 右子树前序”。而中序的序列是"左子树中序 | 根 | 右子树中序"。

因此，我们找一棵子树的根是很简单的，就是前序的第一个元素。然后将左子树和右子树的序列划分出来，递归构造即可。如下：

```C++
TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
    if (preorder.empty()) {
        return nullptr;
    }
    return build(preorder, 0, preorder.size() - 1, inorder, 0, inorder.size() - 1);
}

TreeNode* build(vector<int>& preorder, int preLo, int preHi, 
                vector<int>& inorder, int inLo, int inHi) {
    if (preLo > preHi) {
        return nullptr;
    }

    // 这颗子树的根节点：
    TreeNode* root = new TreeNode(preorder[preLo]);
    // 找左子树、右子树序列
    int inRootIdx = inLo;
    while (inorder[inRootIdx] != preorder[preLo]) {
        inRootIdx++;
    }
    // 计算左右子树序列长度
    int leftLen = inRootIdx - inLo;

    root->left = build(preorder, preLo + 1, preLo + leftLen,
                       inorder, inLo, inLo + leftLen - 1);
    root->right = build(preorder, preLo + leftLen + 1, preHi,
                        inorder, inLo + leftLen + 1, inHi);
    return root;
}
```

为避免在中序中循环查找 root 的索引，我们构建一个 inorder 中值到索引到映射，最终如下：

```C++
class Solution {
public:
    // 存储 inorder 中值到索引的映射
    unordered_map<int, int> valToIndex;

    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        for (int i = 0; i < inorder.size(); i++) {
            valToIndex[inorder[i]] = i;
        }
        return build(preorder, 0, preorder.size() - 1,
                    inorder, 0, inorder.size() - 1);
    }

    // build 函数的定义：
    // 若前序遍历数组为 preorder[preStart..preEnd]，
    // 中序遍历数组为 inorder[inStart..inEnd]，
    // 构造二叉树，返回该二叉树的根节点
    TreeNode* build(vector<int>& preorder, int preStart, int preEnd, 
                    vector<int>& inorder, int inStart, int inEnd) {

        if (preStart > preEnd) {
            return NULL;
        }

        // root 节点对应的值就是前序遍历数组的第一个元素
        int rootVal = preorder[preStart];
        // rootVal 在中序遍历数组中的索引
        int index = valToIndex[rootVal];

        int leftSize = index - inStart;

        // 先构造出当前根节点
        TreeNode* root = new TreeNode(rootVal);
        // 递归构造左右子树
        root->left = build(preorder, preStart + 1, preStart + leftSize,
                        inorder, inStart, index - 1);

        root->right = build(preorder, preStart + leftSize + 1, preEnd,
                        inorder, index + 1, inEnd);
        return root;
    }
};
```

### [106.通过后序和中序遍历结果构造二叉树](https://leetcode.cn/problems/construct-binary-tree-from-inorder-and-postorder-traversal/)

后序 + 中序是能够唯一构造一个二叉树的。与上一题类似，如下：

```C++
// 中序遍历中索引到值的映射
unordered_map<int, int> valToIndex;

TreeNode* build(vector<int>& inorder, int inStart, int inEnd,
                vector<int>& postOrder, int postStart, int postEnd) {
    if (inStart > inEnd) {
        return nullptr;
    }
    int rootVal = postOrder[postEnd];
    int index = valToIndex[rootVal];
    // 左子树节点的个数
    int leftSize = index - inStart;
    TreeNode* root = new TreeNode(rootVal);

    root->left = build(inorder, inStart, index - 1,
                       postorder, postStart, postStart + leftSize - 1);
    root->right = build(inorder, index + 1, inEnd,
                        postorder, postStart + leftSize, postEnd - 1);
    return root;
}

TreeNode* buildTree(vector<int>& inorder, vector<int>& postOrder) {
    for (int i = 0; i < inorder.size(); i++) {
        valToIndex[inorder[i]] = i;
    }
}
```

### [889.根据前序和后续遍历构造二叉树]

已知前序和后续并不能唯一确定一颗二叉树。但这并不影响题目的求解。

- 将前序遍历的第一个元素（或后续遍历的最后一个元素）确定为根节点的值；
- 将前序遍历结果的第二个元素作为左子树的根节点的值；

到这里，看起来好像左右子树唯一，通过preorder第一个元素是根节点，第二个元素是左子树根节点，好像可以唯一确定一颗二叉树，但为什么说不能唯一确定呢？

主要是这一句：

```C++
int leftRootVal = preorder[preStart + 1];
```

默认了前序遍历的第二个元素是左子树的根节点。但实际上左子树有可能是空指针，也就是不存在左子树，这里没办法进行判断，所以导致了最终答案的不唯一。

## 序列化

给出结论：当二叉树中节点值不存在重复时：

- 如果序列化结果不包含空指针信息，且只能给出一种遍历顺序，则无法还原出为一个的一颗二叉树；
- 如果序列化结果中不包含空指针信息，且给出两种遍历顺序：
    1. 如果给出 前序 + 中序 或者 后序 + 中序，那么可以还原出为一的一颗二叉树；
    2. 如果给出前序和后序，无法还原出唯一的一颗二叉树。
- 如果序列化结果中包含空指针信息，且只给出一种遍历顺序：
    1. 如果给出的是前序或后序：可以还原出唯一的一颗二叉树；
    2. 给出的是中序：无法还原出唯一的一颗二叉树。

### [297.二叉树的序列化和反序列化](https://leetcode.cn/problems/serialize-and-deserialize-binary-tree/)

输入一棵树的根节点`root`，要求实现如下类：
```C++
class Codec {
public:
    // 把一棵二叉树序列化成字符串
    string serialize(TreeNode* root);

    // 把字符串反序列化成二叉树
    TreeNode* deserialize(string data);
};
```

通过 `serialize` 方法将二叉树序列化成字符串，用`deserialize`方法将序列化的字符串反序列化成二叉树；

根据上述结论，我们可以有如下解法：

#### 前序遍历解法

通过记录空指针 + 前序遍历的方法解决这个问题。用`,`作为分割符，`#`表示空指针，`serialize`方法如下：
```C++
class Codec {
public:
    string SEP = ",";
    string NULLSYM = "#";

    string serialize(TreeNode* root) {
        string sb;
        _serialize(root, sb);
        return sb;
    }

    // 辅助函数，将二叉树存入 StringBuilder
    void _serialize(TreeNode* root, string& sb) {
        if (root == nullptr) {
            sb.append(NULLSYM).append(SEP);
            return;
        }

        // 前序位置
        sb.append(to_string(root->val)).append(SEP);
        _serialize(root->left, sb);
        _serialize(root->right, sb);
    }
}
```

现在有了 `serialize` 方法，来实现 `deserialize` 函数。

```C++
TreeNode* deserialize(string data) {

    // 由于后面需要 pop_front，所以用 list 来记录
    list<string> nodes;
    istringstream iss(data);
    string s; 
    while (getline(iss, s, ',')) {
        nodes.push_back(s);
    }
    // 然后通过与前序遍历相同的方式构造这棵树
    return _deserialize(nodes);
}

TreeNode* _deserialize(vector<string>& nodes) {
    // base case
    if (nodes.empty()) {
        return nullptr;
    }

    // 前序遍历构造二叉树
    string first = nodes.front();
    nodes.pop_front();

    if (first == NULLSYM) return nullptr;

    TreeNode* root = new TreeNode(stoi(first));
    root->left = _deserialize(nodes);
    root->right = _deserialize(nodes);

    return root;
}
```

#### 后续遍历解法

序列化的过程是很相同的，将`append`过程放在后续即可，如下：

```C++
string SEP = ",";
string NULL_STR = "#";

string serialize(TreeNode* root) {
    string sb;
    _serialize(root, sb);
    return sb;
}

void _serialize(TreeNode* root, string& sb) {
    // 如果根节点是 nullptr，还是 append NULL_STR 即可
    if (root == nullptr) {
        sb.append(NULL_STR).append(SEP);
        return;
    }

    // 后序位置记录，得到的是：左子树 右子树 根
    _serialize(root->left, sb);
    _serialize(root->right, sb);

    // 后序位置
    sb.append(to_string(root->val)).append(SEP);
}
```

对于 deserializer，首先需要找到 root 的值，然后递归计算左右子节点。

root 的值就是最后一个元素。但这里有不同的是，需要先构造 `root.right` 子树，也就是先构造右子树，然后再构造左子树。

因为我们必须先拿到root的值，然后`pop_back()`：此时，向后递归时，先得到的是右子树的根节点（画一个 `string data` 来辅助理解）。总之，不能硬套后序遍历的模板。如下：

```C++
TreeNode* deserialize(string data) {
    // 先读成 list
    istringstream iss(data);
    list<string> nodes;
    string node;
    // getline 最后一个位置必须是 char
    while (getline(iss, node, ',')) {
        nodes.append(node);
    }
    return _deserialize(nodes);
}

TreeNode* _deserialize(list<string>& nodes) {
    if (nodes.empty()) {
        return nullptr;
    }

    // 注意：还是需要先获得根节点
    // _deserialize 本质上还是前序遍历
    string last = nodes.back();
    nodes.pop_back();

    if (last == NULL_STR) return nullptr;
    TreeNode* root = new TreeNode(stoi(last));
    // 注意：这里需要先构造右子树，然后构造左子树
    root->right = _deserialize(nodes);
    root->left = _deserialize(nodes);

    return root;
}
```

中序遍历没法实现 deserialize 方法，因此行不通。

#### 层序遍历解法

我们在每一层后面加入界符，同时需要将 nullptr 记录下来。

反序列化进行构造时，由于记录了 nullptr，我们只需要通过相同的层序遍历模板恢复即可。

序列化和反序列化时，都使用层序遍历的模板，如下：

```C++

                continue;

            sb.append(to_string(cur->val)).append(SEP);
            q.push(cur->left);
            q.push(cur->right);
        }
    }
    return sb;
}
```

## 后序遍历

前序位置的代码只能从函数参数获得父节点传递来的数据，而后序位置的代码不仅可以获取参数数据，还可以获得到子树通过函数返回值传递回来的数据。

换句话说：
- 后序部分的代码能够获得左右子树的信息，然后返回给上一层；
- 一旦题目与“子树”有关，那大概率要给函数设计**合理的定义和返回值**，接受左右子树返回的信息，在后序部分写代码；

### [652.寻找重复的子树](https://leetcode.cn/problems/find-duplicate-subtrees/)

站在每一个节点上，我们都需要知道：
- 以我为根的这颗二叉树长什么样？
- 以其他节点为根的子树都长什么样？

有了序列化子树做铺垫，我们在每一个节点都记录以这个节点为根树的序列化结果，然后判断是否之前出现过相同的子树即可。

为了去重，我们要通过`hashmap<string, int>`来记录这颗子树出现了多少次。如果出现了多次，我们也只记录一次。

注意：serialize 的结果可以是后序或者前序都是可以的。但是不可以是中序遍历。

```C++
// 记录所有子树
unordered_map<string, int> subTrees;
list<TreeNode*> res;

// 将所有子树序列化，并记录最终结果
string serialize(TreeNode* root) {
    if (root == nullptr) {
        return "#";
    }

    string s = std::to_string(root->val);
    s += ",";
    
    s += serialize(root->left); s += ",";
    s += serialize(root->right); s += ",";

    if (subTrees[s] == 1) {
        res.push_back(root);
    }
    subTrees[s]++;
    return s;
}
```
