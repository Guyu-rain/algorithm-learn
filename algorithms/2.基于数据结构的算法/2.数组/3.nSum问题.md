# `nSum` 问题
说白了就是给一个数组`nums`和一个目标`target`，从`nums`选择`n`个数，使这些数的和为`target`。

## 2Sum 问题
最先应该想到的，应该还是遍历。遍历这一颗高度为 2 的排列树，选出最终结果即可，只不过这棵树是用数组表示的。

实际上这种解法就是两层循环，是 $O(n^2)$ 的算法。
```C++
for (int i = 0; i < nums.size(); i++) {
    for (int j = i + 1; j < nums.size(); j++) {
        // 判断 {i, j} 是否成立
    }
}
```
不过，如果数组本身有序（`sort`一下），就可以用双指针解决这个问题。如果结果太大，`hi--`，如果结果太小，`lo++`。这样，总时间复杂度可以控制到`O(nlogn)`。
```C++
vector<int> twoSum(vector<int>& nums, int target) {
    sort(nums.begin(), nums.end());

    int lo = 0, hi = nums.size() - 1;
    while (lo < hi) {
        int sum = nums[lo] + nums[hi];
        if (num < target) {
            lo++;
        } else if (sum > target) {
            hi--;
        } else if (sum == target) {
            return {nums[lo], nums[hi]};
        }
    }
    return {};
}
```
如果不是找一个元素，而是找出所有组，是不是直接在最后一个分支中，将`return`改成存结果就可以呢？还要考虑去重！比如`[1, 1, 1, 3, 3]`, `target=4`，结果应该是`{1, 3}`，但如果直接存结果，那么会出现很多重复元素。也就是应该跳过这些重复元素～

同理：另外两个分支也应该进行去重，如下：
```C++
vector<vector<int>> twoSumTarget(vector<int>& nums, int target) {
    // nums 数组必须有序
    sort(nums.begin(), nums.end());
    int lo = 0, hi = nums.size() - 1;
    vector<vector<int>> res;
    while (lo < hi) {
        int sum = nums[lo] + nums[hi];
        int left = nums[lo], right = nums[hi];
        if (sum < target) {
            while (lo < hi && nums[lo] == left) lo++;
        } else if (sum > target) {
            while (lo < hi && nums[hi] == right) hi--;
        } else {
            res.push_back({left, right});
            while (lo < hi && nums[lo] == left) lo++;
            while (lo < hi && nums[hi] == right) hi--;
        }
    }
    return res;
}
```

## 3Sum 问题
`3Sum = nums[i] + 2Sum`：穷举结论`triple`的的第一个数，然后交给 2Sum 来解决。
```C++
vector<vector<int>> threeSumTarget(vector<int>& nums, int target) {
    // 数组得排个序
    sort(nums.begin(), nums.end());
    int n = nums.size();
    vector<vector<int>> res;
    // 穷举第一个数
    for (int i = 0; i < n; i++) {
        // 对 target - nums[i] 计算 twoSum
            vector<vector<int>> tuples = twoSumTarget(nums, i + 1, target - nums[i]);
            // 如果有满足条件的二元组 tuple，加上 nums[i] 就是三元组
            for (vector<int>& tuple : tuples) {
                tuple.push_back(nums[i]);
                res.push_back(tuple);
            }
        // 跳过第一个数字的重复情况
        while (i < n - 1 && nums[i] == nums[i + 1]) i++;
    }
    return res;
}
// 注意，twoSumTarget 这里多了一个 start 参数，需要从 start 开始向后判断
```

## nSum 问题
如果 `n > 2`，一直递归调用到 `n == 2`；也就是用递归解决 `nSum` 问题
```C++
// 调用之前必须给 nums 排序 !!!
vector<vector<int>> nSumTarget(vector<int>& nums, int n, int start, long target) {
    int sz = nums.size();
    vector<vector<int>> res;

    // 至少是 2Sum, 数组大小不应该小于 n
    if (n < 2 || sz < n) { return res; }

    // base case: 2Sum
    if (n == 2) {
        // 注意：从 start 开始！
        int lo = start, hi = sz - 1;
        while (lo < hi) {
            int left = nums[lo], right = nums[hi];
            int sum = left + right;
            if (sum < target) {
                while (lo < hi && nums[lo] == left) lo++;
            } else if (sum > target) {
                while (lo < hi && nums[hi] == right) hi--;
            } else {
                res.push_back({left, right});
                while (lo < hi && nums[lo] == left) lo++;
                while (lo < hi && nums[hi] == right) hi--;
            }
        }
    } else {
        // 要 n 元组，至少要留 n - 1 个位置
        for (int i = start; i < sz; i++) {
            vector<vector<int>> sub = nSumTarget(nums, n - 1, i + 1, target - nums[i]);
            for (vector<int>& arr : sub) {
                arr.push_back(nums[i]);
                res.push_back(arr);
            }
            // 注意这个地方
            while (i < sz - 1 && nums[i] == nums[i + 1]) { i++; }
            // 此时 nums[i] 是与上一个元素相同的最后一个位置
            // 然后 for 循环还要 i++ 一次，指向了不同的元素
        }
    }
    return res;
}
```
