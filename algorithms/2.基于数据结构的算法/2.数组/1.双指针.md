# 数组中的双指针技巧
详见`1.基础算法`。数组的双指针包括：
- 快慢指针
- 左右指针
这两种。一般用于`数组有序`的情况和`原地操作`的情况.

## 例题
### [LeetCode80.删除有序数组中的重复项II](https://leetcode.cn/problems/remove-duplicates-from-sorted-array-ii/)
本题与单纯删除重复项的区别，是每个元素可以重复出现`2`次。我们用慢指针指向下一个应该放的位置：最终答案为 nums[0..slow]，用`slow`维护一个答案区间。
```C++
int removeDuplicates(vector<int>& nums) {
    if (nums.size() <= 2) {
        return nums.size();
    }
    // 注意，一开始 slow 从 0 开始，nums[0..slow] 中一开始就包含了一个元素
    // 因此 fast 应该从 nums[1] 开始判断
    int slow = 0, fast = 1;
    // 记录 nums[slow] 这个元素个数
    int cnt = 1;
    while (fast < nums.size()) {
        if (nums[slow] != nums[fast]) {
            // 此时，对目标数组而言，nums[fast] 是一个全新的元素
            nums[++slow] = nums[fast];
            cnt = 1;
        } else if (cnt < 2) {
            // 此时 nums[slow] == nums[fast]
            // 但是这个元素的个数还没有超过 2
            nums[++slow] = nums[fast];
            cnt++;
        }

        fast++;
    }
    return size + 1;
}
```
还有一种判断是否可以加入数组的思路，就是考察 `nums[fast]` 是否与 `nums[fast - 2]` 相同。如果相同，则让 `fast` 一直前进到可行的地方，然后将其加入 `nums[0..slow]`
```C++
int removeDuplicates(vector<int>& nums) {
    if (nums.size() <= 2) {
        return nums.size();
    }
    // 前面两个元素不需要判断
    int slow = 2, fast = 2;
    while (fast < nums.size()) {
        // 如果 fast 位置的元素和 slow - 2 个元素不相同，加入答案
        if(nums[fast] != nums[slow - 2]) {
            nums[slow] = nums[fast];
            slow++;
        }

        fast++;
    }
    return slow;
}
```

### [LeetCode125.验证回文串](https://leetcode.cn/problems/valid-palindrome/)
- 回文串的验证：两端向中间靠拢；
- 最长回文串的求解：中间向两端扩散。
本题只需要多一步去除字母外其他字符 + 大写转小写的过程即可。这里只介绍几个常用字符函数：
```C++
#include <cctype>
char c = 'A';
// 字符大写转小写
c = std::tolower(c);
// 整个字符串转小写 - 自定义工具
std::string s = "ABC";
void toLower(string& s) {
    for (char& c : s) {
        c = std::tolower(static_cast<unsigned char>(c));
    }
}

// 字符是否是数字
bool isdigit = std::isdigit(c);
// 是否是字母
bool isalpha = std::isalpha(c);
// 是否是数字或字母
bool isalnum = std::isalnum(c);

// 去掉字符串中非字母数字的其他元素
string keepAlphaNum(const string& s) {
    string res = "";
    for (char c : s) {
        if (std::isalnum(static_cast<unsigned char>(c))) {
            res += c;
        }
    }
    return res;
}
```
一个小注意点：`cctype` 中判断字符的函数，最好传入 `unsigned char`。

### [LeetCode75.颜色分类](https://leetcode.cn/problems/sort-colors/)
**荷兰国旗问题**。最经典的解法是使用`low`, `mid`, `high`三个指针，将数组分`4`段。但每一段的含义必须明确：
```python
[0   ... low - 1] : 全是 0
[low ... mid - 1] : 全是 1
[mid ... high]    : 未处理
[high + 1 ... n ] : 全是 2
```
`low - 1`, `high + 1`也可以写成开区间，实际上就是一开始没有处理，因此区间应该是空。

需要处理的，是 `[mid...high]` 这一段，因此，应该主要考虑 `mid`这个指针：
```C++
void sortColor(vector<int>& nums) {
    int low = 0, mid = 0, high = nums.size() - 1;
    while (mid <= high) {
        // 0(low - 1) 1(low) 1 1(mid - 1) 2(mid) 0 1 1(high) 2(high + 1) 2 2
        if (nums[mid] == 0) {
            std::swap(nums[mid], nums[low]);
            low++;
            mid++; // nums[low..mid] 全是 1
        } else if (nums[mid] == 1) {
            mid++;
        } else {
            std::swap(nums[mid], nums[high])
            high--;
        }
    }
}
```

### [LeetCode88.合并两个有序数组](https://leetcode.cn/problems/merge-sorted-array/)
主要区别是原地。为了不覆盖元素，我们**倒序**遍历`nums1`, `nums2`，这样在覆盖时，前面元素早已用过。
```C++
void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) {
    int i = m - 1, j = n - 1;
    int p = n + n - 1; // 指向 nums1.size() - 1，最后一个位置
    while (i >= 0 && j >= 0) {
        if (nums[i] > nums2[j]) {
            nums[p] = nums[i--];
        } else {
            nums[p] = nums[j--];
        }
        p--;
    }
    // 由于本身就是向 nums[i] 中放元素，如果 nums[i] 没走完，可以直接不管
    while (j >= 0) {
        nums[p--] = nums[j--];
    }
}
```

### [LeetCode1260.二维网格迁移](https://leetcode.cn/problems/merge-sorted-array/)
本题最容易想到的是开一块空间，然后将二维网格映射到一维进行处理。这样是完全可行的；

复习一下矩阵的旋转，可以通过部分多次反转，得到交换顺序（旋转）的结果。本题也同理：
- 将矩阵拉成向量(数组)；
- 将前`m * n - k`个元素反转，后`k`个元素反转；
- 然后将整个数组反转，就得到了正确的反转后的数组；
- 然后将数组转会矩阵即可。
