# 前缀和数组
前缀和技巧用来快速、频繁计算一个区间中所有元素的和。

前缀和数组`preSum[]`有核心公式
```C++
preSum[i] = preSum[i - 1] + nums[i - 1];
```
因此，`nums[i..j]` 这一段从`i -> j`的数组和为：
```C++
preSum[j + 1] - preSum[i]
```
其中，preSum[0] = 0;

## 一维前缀和
完成一个前缀和数组类如下：
```C++
#include <vector>

class NumArray {
    // 前缀和数组
    vector<int> preSum;
public:
    // 输入一个数组，构造前缀和
    NumArray(vector<int>& nums) {
        preSum.resize(nums.size() + 1);
        for (int i = 1; i < preSum.size(); i++) {
            preSum[i] = preSum[i - 1] + nums[i - 1];
        }
    }

    // 查询区间 [left, right] 的累加和
    int sumRange(int left, int right) {
        return preSum[right + 1] - preSum[left];s
    }
};
```

## 二维前缀和
类似的，我们定义：
```C++
preSum[row][col] = nums[row - 1][col - 1] + preSum[row - 1][col] + preSum[row][col - 1] - preSum[row - 1][col - 1];
```
`preSum[row][col]` 记录了 `nums[0..row-1][0..col-1]` 中所有元素的和，可以在`O(1)`求子矩阵`[x1, y1, x2, y2]`中所有元素的和。
```C++
#include <vector>
class NumMatrix {
    vector<vector<int>> preSum;

public:
    // 构造函数
    NumMatrix(vector<vector<int>>& matrix) {
        if (matrix.empty()) { return; }
        int m = matrix.size(), n = matrix[0].size();
        if (m == 0 || n == 0) { return; }
        // 构造前缀和矩阵
        preSum.resize(m + 1, vector<int>(n + 1, 0));
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                preSum[i][j] = matrix[i - 1][j - 1] + preSum[i][j - 1] + preSum[i - 1][j] - preSum[i - 1][j - 1];
            }
        }
    }

    int sumRegion(int x1, int y1, int x2, int y2) {
        return preSum[x2 + 1][y2 + 1] + preSum[x1][y1] - preSum[x1][y2 + 1] - preSum[x2 + 1][y2];
    }
}
```

## 前缀和的局限性
1. 前缀和只适用于 nums 不会变化的情况；
2. 前缀和技巧只适用于存在逆运算的场景。

解决这两个问题，需要**线段树**这种数据结构。

## 例题
### [LeetCode1314.矩阵区域和](https://leetcode.cn/problems/matrix-block-sum/)
实现上述的二维前缀和，然后通过 `min/max` 函数来处理边界情况；
```C++
// NumMatrix 类实现
vector<vector<int>> matrixBlockSum(vector<vector<int>>& mat, int k) {
    int m = mat.size(), n = mat[0].size();
    NumMatrix numMatrix(mat);
    vector<vector<int>> res(m, vector<int>(n, 0));

    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            // 左上角的坐标
            int x1 = max(i - k, 0);
            int y1 = max(j - k, 0);
            // 右下角的坐标
            int x2 = min(i + k, m - 1);
            int y2 = min(j + j, n - 1);

            res[i][j] = numMatrix.sumRegion(x1, y1, x2, y2);
        }
    }
    return res;
}
```

## 前缀积
遇到乘法，就要想是否有除 0 的情况。

### [LeetCode238.除自身以外数组的乘积](https://leetcode.cn/problems/product-of-array-except-self/)
记录一个前缀积和一个后缀积，根据前后缀积就能够求出除了自身其他元素的总积。
```C++
vector<int> productExceptSelf(vector<int>& nums) {
    int n = nums.size();

    // 前缀积与后缀积
    // 注意：为了对称，用 prefix[i] 表示 nums[0..i] 的连乘
    // 用 suffix[i] 表示 nums[i..n-1] 的连乘
    vector<int> prefix(n);
    vector<int> suffix(n);

    // 构造前缀积
    prefix[0] = nums[0]
    for (int i = 1; i < n; i++) {
        prefix[i] = prefix[i - 1] * nums[i];
    }

    // 构造后缀积
    suffix[n - 1] = nums[n - 1];
    for (int i = n - 2; i >= 0; i--) {
        suffix[i] = suffix[i - 1] * nums[i];
    }

    vector<int> res(n);
    res[0] = suffix[1], res[n - 1] = prefix[n - 2];
    for (int i = 1; i < n - 1; i++) {
        res[i] = prefix[i - 1] * suffix[i + 1];
    }
    return res;
}
```

### [LeetCode1352.最后K个数的乘积](https://leetcode.cn/problems/product-of-the-last-k-numbers/)
注意处理
```C++
#include <vector>

class ProductOfNumbers {
    vector<int> preProduct;

public:
    ProductOfNumbers() {
        // 先塞进去一个 1
        preProduct.push_back(1);
    }

    void add(int num) {
        if (num == 0) {
            // 如果添加的元素是 0，前面所有前缀和全都变成 0
            preProduct.clear();
            preProduct.push_back(1);
            return;
        }
        int n = preProduct.size();
        preProduct.push_back(preProduct[n - 1] * num);
    }

    int getProduct(int k) {
        int n = preProduct.size();
        if (k > n - 1) {
            // 遇 0
            return 0;
        }
        // 最后 k 个元素积
        return preProduct[n - 1] / preProduct[n - k - 1];
    }
}
```

### [LeetCode525.连续数组](https://leetcode.cn/problems/contiguous-array/)
如果将`0`换成`-1`，起始就是让求和为 0 的最长子数组。在前缀和数组`preSum`中，寻找`i, j`，使得`preSum[i] - preSum[j] == 0`，且要最大化`j - i`。

我们使用一个 hashMap，存储前缀和到索引的映射，给定`preSum[i]`，都能够通过`valToIndex`这个 hashMap 快速判断是否存在 `j`，使`preSum[i] - preSum[j] == 0`。

```C++
int findMaxLength(vector<int>& nums) {
    int n = nums.size();
    vector<int> preSum(n + 1, 0);
    // 构造前缀和 - 将所有 0 换成 -1
    for (int i = 0; i < n; i++) {
        preSum[i + 1] = preSum[i] + (nums[i] == 0 ? -1 : 1);
    }

    unordered_map<int, int> valToIndex;
    int res = 0;
    for (int i = 0; i < preSum.size(); i++) {
        // 如果这个前缀和还没有对应的索引
        // 说明这个前缀和第一次出现，记录下来
        // 只记录第一次，表示在最前面
        if (valToIndex.find(preSum[i]) == valToIndex.end()) {
            valToIndex[preSum[i]] = i;
        } else {
            // 如果这个前缀和已经出现过，找到一个和为 0 的子数组
            res = max(res, i - valToIndex[preSum[i]]);
        }
    }
    return res;
}
```

### [LeetCode523.连续的子数组和](https://leetcode.cn/problems/continuous-subarray-sum/)
由于数组不变 + 需要某一段的元素和，因此还是用 $O(n)$ 的时间复杂度构造一个前缀和数组(映射：`index -> prefix`)，然后在遍历前缀和数组时，同时存储逆映射：`prefix -> index`，如果能够找到 `prefix[i] - z * k` （其中，$z \in \mathbb{N}$），返回`true`。

考虑同余定理：如果有两个前缀和`prefix[i]`, `prefix[j]` 对 `k` 求余相同，那么`prefix[i] - prefix[j]` 一定是 k 的倍数。因此我们建立 `valToIdx` 这样一个逆映射，用来快速查找**最“远”同余**的子数组。

### [LeetCode560.和为K的子数组](https://leetcode.cn/problems/subarray-sum-equals-k/)
这个“子数组问题”好像可以用滑动窗口来解决？我们来看一下滑动窗口需要回答的三个问题：
1. 什么时候扩大窗口？当窗口内元素 < k 时扩大窗口；
2. 什么时候缩小窗口？当窗口内元素 > k 时缩小窗口；
3. 什么时候返回答案？当窗口内元素 == k 时找到一个答案。
不过，看一下数据范围，是包括负数的！因此上述并不成立。

本题的数组不会变，同时涉及到子数组，因此还是要考虑前缀和。我们有前缀和`prefix`之后，我们很轻松能够得到子数组`prefix[j..i]`中元素之和`prefix[i + 1] - prefix[j]`。

我们可以通过一个`hashMap`来记录`prefix->idx`的映射，不过，本题要求子数组之和为 `k` 的子数组的个数，因此，我们应该记录`prefix[i] -> count`这个映射。也就是记录有多少个前缀和` == prefix[i]`。因此，应该用一个 `count[need]` 来维护 `preSum[0..i]` 中值为`need`的元素个数。

### [LeetCode1124. 表现良好的最长时间段](https://leetcode.cn/problems/longest-well-performing-interval/)
本题中有一个很明显的分界线：`8`。如果有这种线性数据结构 + 一个分界线，考虑将满足条件的全部看成`1`，不满足条件的看成`-1`。

这样做损失信息了吗？并没有！这样做的好处是：我们构造出了“子数组之和”这个重要信息。题目中“表现良好的时间段”等价于 **子数组元素之和 >= 0** 这样一个条件。

至此，题目转换为了“最长的 元素和 >= 0的 子数组”，遇到子数组 + 前缀和，立刻想到使用前缀和数组 + hashMap映射。

```C++
int longestWPI(vector<int>& hours) {
    int n = hours.size();
    vector<int> preSum(n + 1, 0);
    // 前缀和到索引的映射，方便快速查找所需的前缀和
    unordered_map<int, int> valToIndex;
    int res = 0;
    for (int i = 1; i <= n; i++) {
        // 计算 nums[0..i-1] 的前缀和
        preSum[i] = preSum[i - 1] + (hours[i - 1] > 8 ? 1 : -1);
        // 如果这个前缀和还没有对应的索引，说明这个前缀和第一次出现，记录下来
        if (!valToIndex.count(preSum[i])) {
            valToIndex[preSum[i]] = i;
        } else {
            // 因为题目想找长度最大的子数组，valToIndex 中的索引应尽可能小，
            // 所以这里什么都不做
        }

        // 现在我们想找 hours[0..i-1] 中元素和大于 0 的子数组
        // 这就要根据 preSum[i] 的正负分情况讨论了
        if (preSum[i] > 0) {
            // preSum[i] 为正，说明 hours[0..i-1] 都是「表现良好的时间段」
            res = max(res, i);
        } else {
            // preSum[i] 为负，需要寻找一个 j 使得 preSum[i] - preSum[j] > 0
            // 考虑到我们的 preSum 数组每两个相邻元素的差的绝对值都是 1 且 j 应该尽可能小，
            // 那么只要找到 preSum[j] == preSum[i] - 1，nums[j+1..i] 就是一个「表现良好的时间段」
            if (valToIndex.count(preSum[i] - 1)) {
                int j = valToIndex[preSum[i] - 1];
                res = max(res, i - j);
            }
        }
    }
    return res;
}
```
每个位置的`preSum[]`只会被使用一次，如果是新出现的，会被记录在 `valToIndex` 中。因此，`preSum`可以声明成一个`int`，然后 `preSum += nums[i - 1]`这样更新即可。
