# 二维数组的遍历
## 顺/逆时针旋转矩阵（原地）[LeetCode48.旋转图像](https://leetcode.cn/problems/rotate-image/)
> 人类的思维和算法思维常常不同～
直接看解法：先沿着主对角线镜像翻转，然后每一行进行翻转，大概如下：
```python
1 2 3 
4 5 6
7 8 9
```
- 经过主对角线翻转`swap(nums[i][j], nums[j][i])`：
```python
1 4 7
2 5 8
3 6 9
```
- 每一行进行翻转：
```python
7 4 1
8 5 2
9 6 3
```
就变成了顺时针转90度！翻译成代码语言：
```C++
void rotate(vector<vector<int>>& matrix) {
    int n = matrix.size();
    // 先沿着对角线镜像对称二维矩阵
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            // 对角线不用动，因此这里直接 j = i + 1
            std::swap(matrix[i][j], matrix[j][i]);
        }
    }
    // 反转二维矩阵的每一行
    for (auto& row : matrix) {
        std::reverse(row.begin(), row.end());
    }
}
```

特别的，这种“原地反转”，使用“翻转两次”解决还有如下两题：
### 1. [LeetCode151.颠倒字符串中的单词](https://leetcode.cn/problems/reverse-words-in-a-string/)
- 将整个字符转完整翻转；
- 然后再对每个单词反转一次。
不过这个题目需要处理多余空格，并且要维护`string`的迭代器。如下：
```C++
// 首先实现一个去除所有多余空格的函数
// 需要去除前导空格和多余空格
void trimSpaces(string& s) {
    int n = s.size();
    int slow = 0, fast = 0;
    // 去除所有前导空格
    while (s[fast] == ' ') {
        fast++;
    }

    while (fast < n) {
        // 如果遇到一个单词，完整复制下来
        while (fast < n && s[fast] != ' ') {
            s[slow++] = s[fast++];
        }
        // 如果遇到空格，先全部删掉
        while (fast < n && s[fast] == ' ') {
            fast++;
        }
        // 如果后面还有单词，补上一个空格
        if (fast < n && s[fast] != ' ') {
            s[slow++] = ' ';
        }
    }
    // 将 s 只只保留 s[0..(slow-1)]
    s.resize(slow);
}

string reverseWords(string s) {
    // 先去掉所有空格
    trimSpaces(s);
    // 翻转一次
    std::reverse(s.begin(), s.end());
    // 每个词反转一次
    int n = s.size();
    int start = 0;
    for (int i = 0; i <= n; i++) {
        // 当遇到了 ' ' || i == n
        // 也就是出现了一个完整的词，将其翻转
        if(i == n || s[i] == ' ') {
            std::reverse(s.begin() + start, s.begin() + i);
            start = i + 1;
        }
    }
    return s;
}
```

### 2. [LeetCode61.旋转链表](https://leetcode.cn/problems/rotate-list/)
比如链表：`1->2->3->4->5`
要将链表的每个节点向右移动 k 个位置，最朴素的想法是：
1. 找到链表中倒数第`k`个节点，也就是`n-k+1`个节点；
2. 将这个链表为头节点的链表，“摘下来”，拼到链表头。
不过延续“翻转两次”的想法：
1. 整个翻转一次`reverse(head)`;
2. 翻转前`k`个`reverseN(K)`;
3. 翻转后`n-k`个`reverse(newHead)`。
就能得到正确结果。需要注意的是，如果 `k > n`，那么与右移`k % n`的效果是等价的。因此应该先做取余，然后在进行两次翻转。

本题目需要处理边界，按照上面思路能够AC。

## 矩阵的旋转遍历[LeetCode54.旋转矩阵](https://leetcode.cn/problems/spiral-matrix/)
没有什么好方法，不停向某个方向移动，如果遇到了边界，收缩这个边，然后换方向即可。每一个遍历方向都在一个对应的边界（`*_bound`）上进行

不过要额外注意的是，每一个方向（如`从左向右`），都需要判断是否还需要遍历（`upper_bound <= lower_bound`）；
```C++
vector<int> spiralOrder(vector<vector<int>>& matrix) {
    int m = matrix.size(), n = matrix[0].size();
    // “上下”方向
    int upper_bound = 0, lower_bound = m - 1;
    // “左右”方向
    int left_bound = 0, right_bound = n - 1;
    vector<int> res;
    // 用 res 中存了多少个数字判断是否需要结束
    while (res.size() < m * n) {
        // 右移
        if (upper_bound <= lower_bound) {
            for (int j = left_bound; j <= right_bound; j++) {
                res.push_back(matrix[upper_bound][j]);
            }
            // 上边界下移
            upper_bound++;
        }
        // 下移
        if (left_bound <= right_bound) {
            for (int i = upper_bound; i <= lower_bound; i++) {
                res.push_back(matrix[i][left_bound]);
            }
            left_bound--;
        }
        // 左移
        if (upper_bound <= lower_bound) {
            for (int j = right_bound; i >= left_bound; i--) {
                res.push_back(matrix[lower_bound][j]);
            }
            lower_bound--；
        }
        // 上移
        if (left_bound <= right_bound) {
            for (int i = lower_bound; i >= upper_bound; i--) {
                res.push_back(matrix[i][left_bound]);
            }
            left_bound++;
        }
    }
    return res;
}
```
