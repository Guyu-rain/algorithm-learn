# 二分算法
二分算法最烦人的就是边界条件！因此只要一启动二分算法，就直接使用下面的统一框架。什么`>, <`是否取等、什么`mid`是否 `+1, -1` 都不要考虑，直接用下面这个框架。

## 二分查找框架
二分查找有两种写法：
1. 两端都闭，
2. 左闭右开。
下面用两端都闭的写法：
```C++
int binarySearch(vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;

    while (...) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) {
            ...
        } else if (nums[mid] < target) {
            left = ...
        } else if (nums[mid] > target) {
            right = ...
        }
    }
}
```
... 就是需要考虑应该如何更新的地方，需要想清楚如何更新。

这里的分支最好能写`else if`，然后将边界条件写清楚，避免遗漏。

### 搜索区间
需要理解清`left`, `right`这两个指针的含义。

如果初始化`right = nums.size() - 1`，那我们搜索的就是`[left, right]`这样一个两端都闭的区间。反之，如果初始化 `right = nums.size()`，那搜索的就是左闭右开的区间`[left, right)`。

根据搜索区间的不同，对应的写法也不同。对于正确的二分，需要：
1. 搜索区间为空时，应该立刻停止搜索，避免出现死循环；
2. 搜索的过程中，不能漏掉元素，否则会出现错误答案。

下面我们用**两端都闭**的写法，来考虑三种二分的场景：

## 寻找一个数
```C++
int search(vector<int>& nums, int target) {
    int left = 0;
    // 注意
    int right = nums.size() - 1;
    
    // 注意
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) {
            return mid;
        } else if (nums[mid] < target) {
            // 注意：不要重复搜索
            left = mid + 1;
        } else if (nums[mid] > target) {
            // 注意：不要重复搜索
            right = mid - 1;
        }
    }
    return -1;
}
```
两端都闭的搜索区间，需要`<=`避免漏掉元素。同时，由于`nums[mid]`已经搜索过，需要`mid + 1`,`mid - 1`

这种算法的缺点：`nums = [1, 2, 2, 2, 3]`，`target = 2`，此时返回的索引是`2`。但如果想要找`target`左/右侧边界（索引为1/3），是无法处理的。下面讨论这两种算法

### 寻找左侧边界
```C++
int left_bound(vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;
    // 搜索区间为 [left, right]
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] < target) {
            // 搜索区间变为 [mid+1, right]
            left = mid + 1;
        } else if (nums[mid] > target) {
            // 搜索区间变成 [left, mid-1]
            right = mid - 1;
        } else if (nums[mid] == target) {
            // 收缩右侧边界 —— 让左侧边界能指向答案
            // 在区间 [left, mid-1] 中不断搜索，向左收缩，达到锁住左边界的目标
            right = mid - 1;
        }
    }
    return left;
}
```
这个算法返回的是**大于等于target的最小索引**。也就是常见的lower_bound。

如果要求 target 不存在返回 -1，加上边界判断和数值判断即可：
```C++
if (left < 0 || left >= nums.size()) {
    return -1;
}

return nums[left] == target ? left : -1;
```

### 寻找右侧边界
```C++
int right_bound(vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;
    // 搜索区间为 [left, right]
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] < target) {
            left = mid + 1;
        } else if (nums[mid] > target) {
            right = mid - 1;
        } else if (nums[mid] == target) {
            // 收缩左侧边界 —— 让右边界能指向正确答案
            left = mid + 1;
        }
    }
    return right;
}
```
这个算法返回的是**小于等于target的最大索引**，是upper_bound带等于的版本。

如果要求 target 不存在返回 -1，带上如下判断：
```C++
if (right < 0 || right >=nums.size()) {
    return -1;
}

return nums[right] == target ? right : -1;
```

## 左闭右开的写法
最核心的，是要理解搜索区间变成了`[left, right)`。
- 如果 `left == right`，此时已经为空，应该推出循环
- `right`这个位置没有被搜索过，因此需要注意更新`right`时应该如何更新。

```C++
int search(vector<int>& nums, int target) {
    int left = 0, right = nums.size();

    // 搜索区间为 [left, right)
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) {
            return mid;
        } else if (nums[mid] < target) {
            // 将搜索空间收缩到 [mid + 1, right)
            // 不要重复搜索 [left, mid] 这一段
            left = mid + 1;
        } else if (nums[mid] > target) {
            // 将搜索空间收缩到 [left, mid)
            // 注意：mid - 1 这个位置需要被包含到搜索空间中
            right = mid;
        }
    }
    return -1;
}

int left_bound(vector<int>& nums, int target) {
    int left = 0, right = nums.size();

    // 搜索区间为 [left, right)
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] < target) {
            // 逻辑不变
            left = mid + 1;
        } else if (nums[mid] > target) {
            right = mid;
        } else if (nums[mid] == target) {
            // 收缩寻找空间
            right = mid;
        }
    }
    if (left < 0 || left >= nums.size()) {
        return -1;
    }
    return nums[left] == target ? left : -1;
}

int left_bound(vector<int>& nums, int target) {
    int left = 0, right = nums.size();

    // 搜索区间为 [left, right)
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] < target) {
            // 逻辑不变
            left = mid + 1;
        } else if (nums[mid] > target) {
            right = mid;
        } else if (nums[mid] == target) {
            // 收缩寻找空间
            left = mid + 1;
        }
    }
    // 到这里，有 left == right，但 nums[right] 并不合法，其实找到的是前一个元素
    if (left - 1 < 0 || left - 1 >= nums.size()) {
        return -1;
    }
    
    return nums[left - 1] == target ? left - 1 : -1;
}
```
