# 栈的经典习题
## 考察先进后出的性质
### [71.简化路径](https://leetcode.cn/problems/simplify-path/)
不停入栈，每次遇到一个`/`尝试弹出栈内元素：
- 如果遇到了`.`，将其弹出；
- 如果遇到了`..` 将其以及上一级路径一起弹出（弹出到上一个`/`）。

然后压入`/`。直到最后一个元素，然后用另外一个栈来接收这个栈的结果作为最终答案。
```C++
#include <iostream>
#include <vector>
#include <sstream>

string simplifyPath(string path) {
    vector<string> parts;
    istringstream ss(path);
    string part;

    while (getline(ss, part, '/')) {
        if (part.empty() || part == ".") {
            continue;
        }
        if (part == "..") {
            if (!parts.empty()) parts.pop_back();
        } else {
            parts.push_back(part);
        }
    }

    string res;
    for (const auto& p : parts) {
        res += "/" + p;
    }

    return res.empty() ? "/" : res;
}
```

### [20.有效的括号](https://leetcode.cn/problems/valid-parentheses/)
很经典的括号配对，遇到左括号进栈，遇到有括号看是否与栈顶匹配；
```C++
bool isLeft(char c) {
    return c == '(' || c == '[' || c == '{';
}
bool isRight(char c) {
    return c == ')' || c == ']' || c == '}';
}
char leftOf(char c) {
    if (c == ')') return '(';
    if (c == ']') return '[';
    if (c == '}') return '{';
}

bool isValid(string str) {
    stack<char> left;

    for (char& c : str) {
        if (isLeft(c)) {
            left.push(c);
        }
        if (isRight(c)) {
            // 先判空，然后出栈
            if (left.empty()) return false;
            char d = left.top(); d.pop();

            if (d != leftOf(c)) {
                return false;
            }
        }
    }
    // 很容易漏掉，左括号可能没有匹配完
    return left.empty();
}
```

### [150.逆波兰式求值](https://leetcode.cn/problems/evaluate-reverse-polish-notation/)
逆波兰式本身就是通过栈求解的，因此只需要模拟一边过程即可；
```C++
bool isOperator(const string& str) const {
    return str == "+" || str == "-" || str == "*" || str == "/";
}

int evalRPN(vector<string>& tokens) {
    stack<int> stk;

    for (const string& token : tokens) {
        if (isOperator(token)) {
            int op2 = stk.top(); stk.pop();
            int op1 = stk.top(); stk.pop();

            if (token == "+") {
                stk.push(op1 + op2);
            }
            if (token == "-") {
                stk.push(op1 - op2);
            }
            if (token == "*") {
                stk.push(op1 * op2);
            }
            if (token == "/") {
                if (op2 == 0) {
                    throw std::runtime_error("Zero Division error");
                }
                stk.push(op1 / op2);
            }
        } else {
            stk.push(stoi(token));
        }
    }

    return stk.top();
}
```

### [388.文件的最长绝对路径](https://leetcode.cn/problems/longest-absolute-file-path/)
将文件夹视作特殊的文件，先用`\n`split一次，获得所有的“文件”。

我们用一个栈存储按顺序存储文件。栈中存储`tuple<路径名, 路径级, 是否是文件>`。然后用一个`vector<pair<文件路径名, 路径等级>> res`来存储结果，每次出栈都需要遍历vector，看是否需要拼接。最后扫一次`res`，要最长的文件路径名。
```C++
#include <stack>
#include <vector>
#include <string>
#include <sstream>
using namespace std;

int pathRank(string& path) {
    // 从右找 rank
    size_t rank = path.rfind("\t") + 1;
    return rank == string::npos ? 0 : rank;
}

bool isFile(string& path) {
    return path.find('.') != string::npos;
}
int lengthLongestPath(string input) {
    istringstream iss(input);
    // 构造栈
    stack<tuple<string, int, bool>> stk;
    string part;
    while (getline(iss, part, '\n')) {
        int rank = pathRank(part);
        stk.push(make_tuple(part.substr(rank), rank, isFile(part)));
    }
    // 出栈并找结果
    vector<pair<string, int>> res;
    while (!stk.empty()) {
        auto [path, rank, is_file] = stk.top(); stk.pop();
        for (auto& cur : res) {
            auto& [p, r] = cur;
            if (r == rank + 1 && !is_file) {
                p = path + "/" + p;
                r--;
            }
        }
        if (is_file) {
            res.push_back({path, rank});
        }
    }

    int ans = 0;
    for (pair<string, int>& cur : res) {
        ans = std::max(ans, static_cast<int>(cur.first.size()));
    }
    return ans;
}
```
或者正向处理如下：

正向读取一个路径的分隔。`stack` 存储从根目录到当前目录每一层的长度。如果当前层深度`level`小于当前栈的大小，说明已经离开了之前的子目录，回到了上层。
```C++
#include <deque>
#include <string>
#include <vector>
#include <algorithm>
using namespace std;

class Solution {
public:
    int lengthLongestPath(string input) {
        // 这个栈存储之前的父路径。实际上这里只用存父路径的长度就够了，这个优化留给你吧
        deque<int> stack;
        int maxLen = 0;
        vector<string> parts = split(input, '\n');
        for (const string& part : parts) {
            int level = part.rfind("\t") + 1;
            // 让栈中只保留当前目录的父路径
            while (level < stack.size()) {
                stack.pop_back();
            }
            stack.push_back(part.size() - level);
            // 如果是文件，就计算路径长度
            if (part.find('.') != string::npos) {
                int sum = 0;
                for (int len : stack) {
                    sum += len;
                }
                // 加上父路径的分隔符
                sum += stack.size() - 1;
                maxLen = max(maxLen, sum);
            }
        }
        return maxLen;
    }

private:
    vector<string> split(const string& s, char delimiter) {
        vector<string> tokens;
        string token;
        istringstream tokenStream(s);
        while (getline(tokenStream, token, delimiter)) {
            tokens.push_back(token);
        }
        return tokens;
    }
};
```

### [394.字符串解码](https://leetcode.cn/problems/decode-string/)
用一个字符串 `res` 记录最终结果。同时准备两个栈：
- 遇到字符：直接拼接到 `res` 后
- 遇到数字：压入一个单独的栈中（因为数据类型不同）

- 遇到`[`：压入栈中
- 遇到`]`：开始出栈，直到遇到`[`，将其组成一个完整的字符串，从数字栈中弹出一个数字`n`，重复`n`次之后压回栈中。直到栈空，拼接到 res 上。
```C++
string decodeString(string s) {
    stack<string> strStack;
    stack<int> cntStack;

    string cur = "";

    int k = 0;
    for (char c : s) {
        if (isdigit(c)) {
            k = k * 10 + (c - '0');
        } else if (c == '[') {
            // 压栈，重制当前状态
            strStack.push(cur);
            cntStack.push(k);
            cur = "";
            k = 0;
        } else if (c == ']') {
            int times = cntStack.top(); cntStack.pop();
            string prev = strStack.top(); strStack.pop();
            string repeated = "";
            for (int i = 0; i < times; i++) {
                repeated += cur;
            }
            cur = prev + repeated;
        } else {
            cur += c;
        }
    }
    return cur;
}
```
